## 摘要
混合近似最近邻搜索（Hybrid ANNS）是针对大规模异构数据的基础搜索技术，在学术界和工业界都受到了广泛关注。然而，现有方法忽视了数据分布的异构性，从而忽略了两个主要挑战：相似度量级异构性带来的兼容性障碍，以及对属性基数的容忍度瓶颈。为了克服这些问题，我们提出了鲁棒的异构感知混合检索框架 STABLE，旨在为各种分布的数据集提供准确、高效且鲁棒的混合 ANNS。具体而言，我们引入了一种增强型异构语义感知（AUTO）度量，以实现特征相似度和属性一致性的联合测量，解决了相似度量级异构性问题，并提高了对具有不同属性基数的数据集的鲁棒性。随后，我们基于 AUTO 构建了异构语义关系图（HELP）索引，以组织异构语义关系。最后，我们采用了一种新颖的动态异构路由方法来确保高效搜索。在五个具有不同属性基数的特征向量基准测试集上进行的广泛实验证明了 STABLE 的卓越性能。
