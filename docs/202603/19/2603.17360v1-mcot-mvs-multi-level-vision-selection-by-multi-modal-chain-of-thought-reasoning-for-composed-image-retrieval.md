## 摘要
组合图像检索（CIR）旨在根据参考图像和修改文本检索目标图像。然而，现有方法往往难以从参考图像中提取出在文本修改提示下最能反映用户意图的正确语义线索，从而导致受到无关视觉噪声的干扰。在本文中，我们提出了一种用于 CIR 的新型多模态思维链推理多级视觉选择（MCoT-MVS）方法，该方法集成了由多模态大语言模型（MLLM）推理线索引导的注意力感知多级视觉特征。具体而言，我们利用 MLLM 对多模态组合输入进行思维链推理，生成保留、移除和目标推断的文本。这些文本线索随后引导两个参考视觉注意力选择模块，从参考图像中选择性地提取具有辨别力的补丁级（patch-level）和实例级（instance-level）语义。最后，为了有效地将这些多粒度视觉线索与修改后的文本及想象的目标描述融合，我们设计了一个加权分层组合模块，在统一的嵌入空间中将组合查询与目标图像对齐。在两个 CIR 基准数据集（即 CIRR 和 FashionIQ）上的广泛实验表明，我们的方法始终优于现有方法，并达到了新的最先进性能。代码和训练模型已公开。

## Abstract
Composed Image Retrieval (CIR) aims to retrieve target images based on a reference image and modified texts. However, existing methods often struggle to extract the correct semantic cues from the reference image that best reflect the user's intent under textual modification prompts, resulting in interference from irrelevant visual noise. In this paper, we propose a novel Multi-level Vision Selection by Multi-modal Chain-of-Thought Reasoning (MCoT-MVS) for CIR, integrating attention-aware multi-level vision features guided by reasoning cues from a multi-modal large language model (MLLM). Specifically, we leverage an MLLM to perform chain-of-thought reasoning on the multimodal composed input, generating the retained, removed, and target-inferred texts. These textual cues subsequently guide two reference visual attention selection modules to selectively extract discriminative patch-level and instance-level semantics from the reference image. Finally, to effectively fuse these multi-granular visual cues with the modified text and the imagined target description, we design a weighted hierarchical combination module to align the composed query with target images in a unified embedding space. Extensive experiments on two CIR benchmarks, namely CIRR and FashionIQ, demonstrate that our approach consistently outperforms existing methods and achieves new state-of-the-art performance. Code and trained models are publicly released.

*   **研究背景**：组合图像检索（CIR）要求系统根据“参考图像”和“修改文本”检索目标图像。
*   **核心问题**：现有方法在处理复杂的修改指令时，往往难以准确区分参考图像中哪些部分应保留、哪些应删除。这导致检索模型容易受到无关视觉噪声的干扰（例如，文本要求删除某物体，但模型因视觉相似性仍关注该物体），缺乏对用户意图的显式推理能力。
*   **整体含义**：本文提出 **MCoT-MVS** 框架，旨在通过多模态大语言模型（MLLM）的思维链（CoT）推理来分解用户意图，并以此引导模型在“补丁级”和“实例级”两个层面显式选择视觉特征，从而实现更精准的语义对齐。

### 2. 方法论
核心架构分为三个阶段：
*   **多模态思维链推理（MCoT Reasoning）**：
    *   利用预训练 MLLM（如 Qwen2.5-VL）执行四步推理：描述参考图、解释修改指令、推断目标图描述、识别需“保留”和“删除”的视觉元素。
    *   输出三种文本线索：保留文本（$RT$）、删除文本（$DT$）和目标推断文本（$TT$）。
*   **多级视觉参考选择（PVRS & IVRS）**：
    *   **补丁级选择（PVRS）**：计算图像 Patch 与 $RT$、$DT$ 的余弦相似度，通过正负反馈机制重构 Patch 权重，抑制噪声区域。
    *   **实例级选择（IVRS）**：利用 Grounded SAM 提取图像中的具体实例，根据推理出的保留/删除意图对实例特征进行加权聚合，捕捉高层语义和形状信息。
*   **加权分层组合模块（WHC）**：
    *   设计了一个自适应组合器，将选择后的视觉特征与原始修改文本、推理出的目标文本进行分层融合。通过注意力机制动态分配权重，生成最终的统一查询向量。

### 6. 主要结论与发现
*   **性能领先**：MCoT-MVS 在 CIRR 和 FashionIQ 上均刷新了 SOTA 记录。在 CIRR 上的平均 Recall 达到 82.52%，在 FashionIQ 上的平均 R@10 达到 63.24%。
*   **推理的重要性**：显式的意图分解（保留 vs 删除）比直接融合多模态特征更有效。
*   **多级协同**：补丁级特征提供细粒度细节，实例级特征提供完整物体语义，两者结合能显著提升检索精度。

### 7. 优点
*   **可解释性强**：通过 MCoT 生成的文本和视觉注意力图，可以清晰观察到模型对用户意图的理解过程。
*   **多粒度融合**：结合了自底向上的 Patch 特征和自顶向下的实例特征，解决了 CIR 中常见的语义偏移问题。
*   **即插即用潜力**：框架逻辑清晰，可以适配不同的 MLLM 和视觉骨干网络。

### 8. 不足与局限
*   **推理延迟**：引入 MLLM 和 Grounded SAM 显著增加了单次查询的计算开销，对于实时性要求极高的工业场景可能存在挑战。
*   **幻觉风险**：MLLM 在推理目标文本时可能产生“幻觉”（生成图中不存在的属性），虽然 WHC 模块有过滤机制，但仍可能干扰最终表征。
*   **依赖预训练模型**：性能高度依赖于底层 MLLM 的推理能力和 SAM 的分割精度，在极端长尾分布的数据上可能受限。
