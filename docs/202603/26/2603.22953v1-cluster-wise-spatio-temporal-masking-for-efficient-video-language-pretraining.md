## 摘要
大规模视频-语言预训练在多模态任务中实现了强大的泛化能力，但往往伴随着极高的计算成本。尽管掩码视觉建模（masked visual modeling）的最新进展有助于缓解这一问题，但仍面临两个根本性局限：高掩码率下的严重视觉信息丢失，以及由帧间相关性引起的时间信息泄漏。为应对这些挑战，我们提出了 ClusterSTM，一种用于高效视频-语言预训练的基于聚类的时空掩码策略。ClusterSTM 首先进行帧内聚类，将视觉 token 划分为多个语义独立的聚类，然后通过保留每个聚类中具有最高时间密度的 token 来执行基于聚类的掩码。我们的掩码策略确保保留的 token 在捕捉整体视频内容的同时，展现出强的时间相关性。此外，我们引入了一种视频-文本相关性重构目标，在传统的视觉重构之外对齐高层多模态语义。在多个基准测试上的广泛实验表明，ClusterSTM 在视频-文本检索、视频问答和视频字幕生成任务中取得了卓越的性能，在高效视频-语言模型中达到了新的最先进水平。

## Abstract
Large-scale video-language pretraining enables strong generalization across multimodal tasks but often incurs prohibitive computational costs. Although recent advances in masked visual modeling help mitigate this issue, they still suffer from two fundamental limitations: severe visual information loss under high masking ratios and temporal information leakage caused by inter-frame correlations. To address these challenges, we propose ClusterSTM, a Cluster-Wise Spatio-Temporal Masking strategy for efficient video-language pretraining. ClusterSTM first performs intra-frame clustering to partition visual tokens into multiple semantically independent clusters, then conducts cluster-wise masking by retaining the token with the highest temporal density within each cluster. Our masking strategy ensure that the retained tokens capture holistic video content while exhibit strong temporal correlation. Additionally, we introduce a video-text relevance reconstruction objective that aligns high-level multimodal semantics beyond conventional visual reconstruction. Extensive experiments across multiple benchmarks demonstrate that ClusterSTM achieves superior performance on video-text retrieval, video question answering, and video captioning tasks, establishing a new state-of-the-art among efficient video-language models.

以下是对该论文的结构化深入总结：

### 1. 核心问题与研究动机
*   **核心问题**：大规模视频-语言预训练虽然泛化能力强，但计算开销巨大。现有的掩码视觉建模（MVM）技术在应用于视频时存在两个痛点：
    1.  **信息丢失严重**：在高掩码率（如 90%）下，随机掩码容易丢失关键的视觉语义（包括前景和背景）。
    2.  **时间信息泄露**：视频帧间存在高度冗余，简单的掩码策略（如随机掩码）使得模型可以通过相邻帧轻松“偷看”到被掩码的内容，导致表征学习效率低下。
*   **研究背景**：现有的方法如 UMT 侧重于语义掩码（只看前景），而 STM 虽然降低了掩码率但牺牲了效率。本文旨在寻找一种既能保持高掩码率（高效率），又能捕捉完整语义并克服时间冗余的方案。

### 2. 核心方法论
ClusterSTM 包含两个关键创新组件：

*   **基于聚类的时空掩码（Cluster-Wise Spatio-Temporal Masking）**：
    1.  **帧内聚类（Intra-Frame Clustering）**：使用密度峰值聚类（DPC）算法将每帧的视觉 Token 划分为多个语义独立的簇。这确保了掩码过程能够感知到不同的语义区域（如人、背景、物体）。
    2.  **时间密度筛选**：计算每个 Token 与相邻帧所有 Token 的语义距离，定义“时间密度”。密度越高，代表该 Token 在时间维度上的语义一致性越强。
    3.  **簇内保留**：在每个聚类簇中，仅保留时间密度最高的那个 Token，其余全部掩码。这种策略确保了保留的 Token 既能代表该簇的语义，又具有跨帧的稳定性，有效防止了因帧间偏移导致的信息泄露。
*   **视频-文本相关性重构（Video-Text Relevance Reconstruction, MRM）**：
    *   不同于传统的像素或视觉特征重构，本文引入了**多模态重构目标**。
    *   利用教师模型（如 SigLIP）生成视频 Token 与文本特征的细粒度相关性矩阵。学生模型需要根据未掩码的 Token 重构出这些跨模态的相关性，从而在预训练阶段就强化高层语义的对齐。

### 6. 主要结论与发现
*   **性能领先**：在 5M 预训练规模下，ClusterSTM 在所有测试任务中均刷新了 SOTA 记录。例如，在 MSRVTT 检索任务上 Recall@1 达到 49.7%，显著超过了之前的 STM 和 UMT。

*   **掩码策略的有效性**：实验证明，基于聚类的保留策略比单纯的语义掩码（只看前景）更全面，因为它保留了对跨模态对齐至关重要的背景上下文。

### 7. 优点与亮点
*   **创新的掩码视角**：将聚类（保证空间覆盖）与时间密度（保证时间一致性）结合，巧妙解决了视频数据特有的冗余问题。
*   **多模态重构目标**：MRM 损失函数将重构任务从纯视觉维度提升到了跨模态语义维度，更符合视频-语言任务的本质。
*   **极高的计算效率**：在 90% 的极高掩码率下依然保持高性能，极大地缩短了预训练周期并降低了显存占用。
