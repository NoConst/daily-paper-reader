## 摘要
多模态大语言模型（MLLMs）在文档理解和密集场景感知等细粒度任务中需要高分辨率视觉输入。然而，当前的全局分辨率缩放范式不加区分地将视觉冗余的 token 注入具有二次复杂度的自注意力机制中，在忽略空间稀疏性和查询意图的同时，严重制约了推理吞吐量。为解决这一问题，我们提出了 Q-Zoom，这是一个以高效的“由粗到精”方式运行的查询感知自适应高分辨率感知框架。首先，当粗粒度的全局特征足够时，轻量级动态门控网络（Dynamic Gating Network）会安全地跳过高分辨率处理。其次，对于需要细粒度感知的查询，自蒸馏区域建议网络（SD-RPN）直接从中间特征空间精确地定位与任务相关的感兴趣区域（RoI）。为了高效优化这些模块，门控网络使用一致性感知生成策略来推导确定性的路由标签，而 SD-RPN 则采用完全自监督的蒸馏范式。随后，通过连续的时空对齐方案和针对性微调，将密集的局部 RoI 与粗粒度的全局布局无缝融合。大量实验表明，Q-Zoom 建立了领先的帕累托前沿。以 Qwen2.5-VL-7B 作为主要测试平台，Q-Zoom 在文档与 OCR 基准测试中将推理速度提升了 2.52 倍，在高分辨率场景中提升了 4.39 倍，同时达到了基线的峰值准确率。此外，当配置为最大感知保真度时，Q-Zoom 在这些基准测试中分别超过基线峰值性能 1.1% 和 8.1%。这些显著的改进可以无缝迁移到 Qwen3-VL、LLaVA 以及新兴的基于强化学习（RL）的“带图思考”模型中。项目页面见 https://yuhengsss.github.io/Q-Zoom/。

## Abstract
MLLMs require high-resolution visual inputs for fine-grained tasks like document understanding and dense scene perception. However, current global resolution scaling paradigms indiscriminately flood the quadratic self-attention mechanism with visually redundant tokens, severely bottlenecking inference throughput while ignoring spatial sparsity and query intent. To overcome this, we propose Q-Zoom, a query-aware adaptive high-resolution perception framework that operates in an efficient coarse-to-fine manner. First, a lightweight Dynamic Gating Network safely bypasses high-resolution processing when coarse global features suffice. Second, for queries demanding fine-grained perception, a Self-Distilled Region Proposal Network (SD-RPN) precisely localizes the task-relevant Region-of-Interest (RoI) directly from intermediate feature spaces. To optimize these modules efficiently, the gating network uses a consistency-aware generation strategy to derive deterministic routing labels, while the SD-RPN employs a fully self-supervised distillation paradigm. A continuous spatio-temporal alignment scheme and targeted fine-tuning then seamlessly fuse the dense local RoI with the coarse global layout. Extensive experiments demonstrate that Q-Zoom establishes a dominant Pareto frontier. Using Qwen2.5-VL-7B as a primary testbed, Q-Zoom accelerates inference by 2.52 times on Document & OCR benchmarks and 4.39 times in High-Resolution scenarios while matching the baseline's peak accuracy. Furthermore, when configured for maximum perceptual fidelity, Q-Zoom surpasses the baseline's peak performance by 1.1% and 8.1% on these respective benchmarks. These robust improvements transfer seamlessly to Qwen3-VL, LLaVA, and emerging RL-based thinking-with-image models. Project page is available at https://yuhengsss.github.io/Q-Zoom/.

### 论文总结：Q-Zoom —— 面向高效多模态大语言模型的查询感知自适应感知框架

#### 1. 核心问题与整体含义（研究动机和背景）
多模态大语言模型（MLLM）在处理文档理解、OCR和密集场景感知等任务时，极度依赖高分辨率输入。然而，现有的主流方法（如 AnyRes 或动态分辨率缩放）存在两个严重缺陷：
*   **查询冗余（Query Redundancy）**：不加区分地对所有图像进行高分辨率处理，即使是简单的问题（如“图中有什么颜色”）也会消耗大量计算资源。
*   **空间冗余（Spatial Redundancy）**：全局缩放会产生数千个视觉 token，其中大部分属于无关背景，导致 LLM 的自注意力机制出现计算瓶颈。
**Q-Zoom 的核心目标**是实现“由粗到精”的自适应感知，仅在必要时、针对必要的区域进行高分辨率处理，从而在不损失精度的前提下大幅提升推理效率。

#### 2. 方法论：核心思想与关键技术
Q-Zoom 提出了一个两阶段框架，直接在模型的中间特征空间运行：
*   **动态门控网络（Dynamic Gating Network）**：
    *   **核心思想**：预测当前查询是否需要高分辨率细节。
    *   **技术细节**：利用 LLM 中间层特征，通过一个轻量级分类头输出概率。
    *   **一致性感知训练**：通过比较模型在不同分辨率下的回答一致性来自动生成训练标签（若高分辨率对对、低分辨率错，则标记为“需要细化”），无需人工标注。
*   **自蒸馏区域建议网络（SD-RPN）**：
    *   **核心思想**：定位与查询相关的感兴趣区域（RoI）。
    *   **技术细节**：复用 LLM 中间层的交叉注意力图（Cross-Attention Maps），通过去除“汇聚 Token”（Sink Tokens）和三态标签分配（确定前景、背景和忽略区）来生成高质量伪标签进行自监督训练。
*   **时空对齐与针对性微调（Post-SFT）**：
    *   **核心思想**：解决局部 RoI 与全局布局的空间错位。
    *   **技术细节**：引入连续时空位置编码，利用插值技术将高分辨率 RoI 映射回原始坐标系。通过挖掘“基础模型对、RoI 模型错”的硬样本进行微调，增强模型融合全局与局部信息的能力。

*   **数据集与场景**：
    *   **文档与 OCR**：DocVQA, InfoVQA, ChartQA, OCRBench, TextVQA。

    *   **通用能力**：MMStar, MME。

*   **对比方法**：包括训练无关方法（ViCrop）、分辨率缩放（S²）、以及最新的 RL 驱动模型（Thyme, DeepEyes, AdaptVision）。

*   **实验规模**：论文涵盖了从 3B 到 13B 不同规模的多个模型家族，并在超过 10 个主流基准测试上进行了验证。

    *   各组件（RPN, SFT, Gate）的有效性。
    *   中间层深度（B）和 RPN 层数（R）的影响。

#### 6. 主要结论与发现
*   **性能领先**：Q-Zoom 在 Qwen2.5-VL-7B 上，相比 4096 token 的暴力基线，在 Token 数量减少 53%-73% 的情况下，准确率反而提升了 1.1%-8.1%。
*   **推理加速**：在文档任务中实现 **2.52 倍**加速，在高分辨率任务中实现 **4.39 倍**加速。
*   **帕累托前沿**：Q-Zoom 在“精度-效率”权衡曲线上显著优于所有现有方法（包括最新的 RL 模型）。
*   **通用性**：该框架可作为插件应用于 RL 训练的模型，并进一步提升其性能。

#### 7. 优点（亮点）
*   **高效性**：在单次预填充（Prefilling）过程中完成决策和定位，避免了 RL 方法中冗长的思维链（CoT）解码延迟。
*   **无需人工标注**：通过自蒸馏和一致性感知策略，完全利用模型自身能力生成训练信号。
*   **即插即用**：能够无缝集成到多种 MLLM 架构中，且对模型基础能力的损伤极小。
*   **空间对齐**：通过连续位置编码解决了局部裁剪带来的空间逻辑混乱问题。

#### 8. 不足与局限
*   **架构依赖**：该方法依赖于模型中间层具备良好的视觉接地（Visual Grounding）能力，对于某些视觉编码能力极弱的基座模型可能效果受限。
*   **硬样本挖掘成本**：Post-SFT 阶段需要使用更强大的模型（如 Qwen2.5-VL-7B-Instruct）作为裁判来筛选样本，这增加了一定的离线数据准备成本。
*   **极端场景**：如果动态门控网络出现误判（将复杂查询误认为简单查询），模型将无法触发高分辨率分支，导致细粒度信息丢失。
