## 摘要
现有的提高大视觉语言模型（LVLM）效率的方法主要基于视觉标记（token）减少的概念。然而，这种方法会产生信息瓶颈，从而损害性能，特别是在需要细粒度理解和推理的挑战性任务上。在这项工作中，我们通过引入 VISion On Request (VISOR) 来挑战这一范式，这是一种在不丢弃视觉信息的情况下降低推理成本的方法。VISOR 并非压缩图像，而是通过稀疏化图像和文本标记之间的交互来提高效率。具体而言，语言模型通过一组少量且经过策略性布置的注意力层来关注全套高分辨率视觉标记：通用的视觉上下文由文本-图像之间的高效交叉注意力提供，而少数位置合理且动态选择的自注意力层则负责细化视觉表示本身，从而在需要时实现复杂的、高分辨率的推理。基于这一原理，我们首先通过改变自注意力层的数量，在不同的计算预算范围内训练一个单一的通用网络，然后引入一种轻量级的策略机制，根据每个样本的复杂度动态分配视觉计算资源。大量实验表明，VISOR 在大幅降低计算成本的同时，在各种基准测试中达到或超过了最先进的结果，并且在需要详细视觉理解的挑战性任务中表现出色。

## Abstract
Existing approaches for improving the efficiency of Large Vision-Language Models (LVLMs) are largely based on the concept of visual token reduction. This approach, however, creates an information bottleneck that impairs performance, especially on challenging tasks that require fine-grained understanding and reasoning. In this work, we challenge this paradigm by introducing VISion On Request (VISOR), a method that reduces inference cost without discarding visual information. Instead of compressing the image, VISOR improves efficiency by sparsifying the interaction between image and text tokens. Specifically, the language model attends to the full set of high-resolution visual tokens through a small, strategically placed set of attention layers: general visual context is provided by efficient cross-attention between text-image, while a few well-placed and dynamically selected self-attention layers refine the visual representations themselves, enabling complex, high-resolution reasoning when needed. Based on this principle, we first train a single universal network on a range of computational budgets by varying the number of self-attention layers, and then introduce a lightweight policy mechanism that dynamically allocates visual computation based on per-sample complexity. Extensive experiments show that VISOR drastically reduces computational cost while matching or exceeding state-of-the-art results across a diverse suite of benchmarks, and excels in challenging tasks that require detailed visual understanding.

### 1. 核心问题与整体含义（研究动机和背景）
大型视觉语言模型（LVLM）在处理高分辨率图像时，由于视觉标记（Token）数量庞大，推理成本极高。现有的主流优化方案（如 Token 剪枝或合并）虽然能降低计算量，但会造成不可逆的**信息瓶颈**，导致模型在需要细粒度理解的任务（如 OCR、文档分析、复杂推理）上性能大幅下降。
本文提出了 **VISOR (VISion On Request)**，其核心动机是：**与其减少视觉 Token 的数量，不如稀疏化视觉与语言之间的交互频率**。作者观察到，并非所有任务都需要在每一层进行深度的图文交互，因此主张根据任务复杂度动态调用视觉计算资源。

### 2. 论文提出的方法论
VISOR 通过解耦文本和视觉的处理流，实现了高效的推理：
*   **核心思想**：将 LLM 的层分为三类：
    1.  **纯文本层**：仅处理文本 Token，不与视觉信息交互。
    2.  **交叉注意力层 (CA)**：文本 Token 作为 Query 去查询静态的视觉特征。这种方式成本极低，因为它不更新视觉 Token 的表示。
    3.  **自注意力层 (SA)**：执行完整的图文混合自注意力，允许模型更新和细化视觉特征。这是高成本操作，仅在少数关键层执行。
*   **关键技术细节**：
    *   **通用模型训练 (Universal Training)**：训练一个单一模型，使其在训练过程中随机采样不同的 SA 层配置，从而能够稳健地支持从“零 SA 层”到“全量 SA 层”的多种计算预算。
    *   **动态路由机制 (Adaptive Inference)**：引入一个轻量级的 MLP 策略网络（Router），在推理时根据输入的指令和图像特征，预测该样本所需的最佳 SA 层配置。
    *   **位置编码优化**：采用 1D 深度卷积层实现条件位置嵌入，以保留高分辨率图像的空间信息。

### 6. 主要结论与发现
*   **性能与效率双赢**：VISOR 在 Hard 任务上显著优于现有的 Token 压缩方法，同时在 Easy 任务上保持了极高的效率（最高节省 8.6 倍 FLOPs）。
*   **视觉特征演化**：通过 CKA 相似度分析发现，简单任务的视觉特征在层间几乎不变，而复杂任务则需要通过 SA 层进行逐层细化。
*   **正交性**：VISOR 可以与 Token 缩减方法结合（VISOR-TR），在损失极小精度的情况下实现高达 18x-35x 的计算节省。
*   **路由准确性**：动态路由机制能准确识别样本难度，为简单样本分配极少计算量，为复杂样本保留深度推理能力。

### 7. 优点（亮点）
*   **范式创新**：打破了“必须减少 Token 数量才能提效”的思维定式，通过稀疏交互保留了完整的视觉上下文。
*   **动态适配**：实现了真正的“按需分配”计算资源，比固定压缩率的方法更具灵活性。
*   **通用性强**：一个模型即可适配多种计算预算，且在不同规模的 LLM 底座上均表现稳健。

### 8. 不足与局限
*   **内存占用 (KV Cache)**：虽然 VISOR 减少了计算量 (FLOPs)，但由于保留了全量视觉 Token，其 KV Cache 的内存占用在某些层可能依然较高（尽管 CA 层不更新视觉特征，但文本 Token 的 KV 增长依然存在）。
*   **路由训练依赖伪标签**：策略网络的训练依赖于离线生成的伪标签，这可能限制了其在完全未见过的分布外的泛化上限。
*   **架构修改**：相比于纯训练无关的剪枝方法，VISOR 需要对模型架构进行微调和训练，部署门槛略高。
