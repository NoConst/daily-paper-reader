## 摘要
从长期的多模态网页交互历史中检索相关观测结果具有挑战性，因为相关性取决于不断演变的任务状态、模态（截图、HTML 文本、结构化信号）以及时间距离。先前的方法通常依赖于静态相似度阈值或固定容量的缓冲区，无法使相关性适应当前的任务上下文。我们提出了 ACGM，一种学习型图记忆检索器，它利用下游任务成功的策略梯度优化，在智能体历史记录上构建任务自适应的相关性图。ACGM 通过特定模态的衰减（视觉衰减比文本快 4.3 倍：λv=0.47 对比 λx=0.11）捕捉异构的时间动态，并学习稀疏连接（每节点 3.2 条边），从而实现高效的 O(log T) 检索。在 WebShop、VisualWebArena 和 Mind2Web 数据集上，ACGM 将检索质量提升至 82.7 nDCG@10（相比 GPT-4o 提升 9.3，p < 0.001）和 89.2% Precision@10（提升 7.7），优于 19 个强劲的稠密检索、重排序、多模态及基于图的基准模型。

## Abstract
Retrieving relevant observations from long multi-modal web interaction histories is challenging because relevance depends on the evolving task state, modality (screenshots, HTML text, structured signals), and temporal distance. Prior approaches typically rely on static similarity thresholds or fixed-capacity buffers, which fail to adapt relevance to the current task context. We propose \textbf{ACGM}, a learned graph-memory retriever that constructs \emph{task-adaptive} relevance graphs over agent histories using policy-gradient optimization from downstream task success. ACGM captures heterogeneous temporal dynamics with modality-specific decay (visual decays $4.3\times$ faster than text: $λ_v{=}0.47$ vs.\ $λ_x{=}0.11$) and learns sparse connectivity (3.2 edges/node), enabling efficient $O(\log T)$ retrieval. Across WebShop, VisualWebArena, and Mind2Web, ACGM improves retrieval quality to \textbf{82.7 nDCG@10} (+9.3 over GPT-4o, $p{<}0.001$) and \textbf{89.2\% Precision@10} (+7.7), outperforming 19 strong dense, re-ranking, multi-modal, and graph-based baselines. *   **研究背景**：网页智能体（Web Agents）在执行长程任务（如在线购物、复杂导航）时，会产生包含截图、HTML文本和结构化数据的海量历史记录。
*   **核心痛点**：
    1.  **动态相关性**：历史信息的相关性随任务进度变化（例如，比价时的图片在支付阶段可能不再重要）。
    2.  **模态异构性**：视觉和文本信息的记忆衰减速度不同，传统方法采用统一的时间衰减权重。
    3.  **效率瓶颈**：随着历史增长，线性扫描所有状态会导致实时性下降，而固定容量缓存又容易丢失远期关键信息。
*   **研究动机**：开发一种能够根据任务反馈自动调整检索策略、识别不同模态时间特性、且具备亚线性检索效率的记忆系统。

### 2. 方法论：ACGM (Adaptive Cross-Modal Graph Memory)
*   **核心思想**：将检索建模为一种**任务自适应的图构建过程**。通过策略梯度（Policy Gradient）学习一个神经预测器，根据下游任务的成功与否来优化历史观测值之间的连接权重。
*   **关键技术细节**：
    *   **学习型相关性预测器 ($g_\phi$)**：一个2层MLP，输入为多模态嵌入（CLIP/RoBERTa）、时间距离和余弦相似度，输出两点间存在相关性边的概率。
    *   **策略梯度优化**：使用 REINFORCE 算法，以任务最终的成功（1）或失败（0）作为奖励信号。引入指数移动平均（EMA）作为基准线（Baseline）以降低梯度方差。
    *   **模态特定时间衰减**：为视觉 ($v$)、文本 ($x$) 和知识 ($k$) 模态分别学习衰减率 $\lambda_m$。公式通过时间加权注意力机制聚合邻居节点信息。
    *   **两层分级检索结构**：
        *   **近期层**：最近20步采用扁平存储。
        *   **远期层**：使用在线 K-means 聚类构建 4 叉树，实现 $O(\log T)$ 的检索复杂度。
*   **训练协议**：采用两阶段训练。第一阶段使用专家演示进行监督预训练（学习基础相似度）；第二阶段进行端到端任务自适应微调。

*   **性能领先**：ACGM 在 WebShop 上达到 82.7 nDCG@10，比 GPT-4o 高出 9.3 个点。
*   **认知一致性**：实验发现视觉记忆衰减速度（$\lambda_v=0.47$）比文本（$\lambda_x=0.11$）快约 4.3 倍，这与认知科学的研究结论一致。

### 7. 优点
*   **任务自适应**：突破了传统静态相似度的限制，使检索系统能根据“是否有助于任务成功”来进化。
*   **多模态精细化**：首次在智能体检索中量化并应用了不同模态的异构时间衰减特性。
*   **扩展性强**：分级树结构和稀疏图设计解决了长序列历史下的计算开销问题。

### 8. 不足与局限
*   **奖励稀疏性**：依赖二元任务成功信号，对于步骤极多（如数百步以上）且中间反馈极少的任务，策略梯度的信用分配（Credit Assignment）可能变得困难。
