## 摘要
视觉语言模型（VLMs）在指令遵循和开放式视觉语言推理方面表现出强大的性能，但它们经常生成与视觉证据关联较弱的流畅输出。先前的研究表明，指令提示会通过放大语言先验进一步恶化这一问题，尤其是在视觉信号不确定或模糊时。为了应对这一挑战，我们提出了一个在生成过程中显式平衡语言信息量和视觉忠实度的解码框架。我们的方法，即指令-证据对比双流解码（IECD2），在每个解码步骤中维持两个并行的 token 概率分布：一个促进表达性和信息丰富响应的指令驱动流，以及一个强制严格基于图像证据的证据驱动流。这两个流使用基于对称 KL 散度的对比门控进行自适应融合，该门控抑制了受语言先验青睐但缺乏视觉证据支持的 token，同时在两个分布一致时保留它们。我们在涵盖各种生成式视觉语言推理任务（如图像描述和视觉问答）的多个数据集上评估了 IECD2，包括 POPE、MME、VQAv2、AMBER、MS-COCO 和 LLaVA-Bench。与最先进的解码方法相比，IECD2 在任务准确性和推理性能方面表现出持续的改进，同时在所有评估指标上显著减少了幻觉。

## Abstract
Vision-Language Models (VLMs) exhibit strong performance in instruction following and open-ended vision-language reasoning, yet they frequently generate fluent outputs that are weakly grounded in visual evidence. Prior works have shown that instruction prompting further worsens this issue by amplifying language priors, especially when the visual signal is uncertain or ambiguous. To address this challenge, we propose a decoding framework that explicitly balances linguistic informativeness and visual faithfulness during generation. Our method, Instruction-Evidence Contrastive Dual-Stream Decoding (IECD2), maintains two parallel probability distributions of tokens at each decoding step: an instruction-driven stream that promotes expressive and informative responses, and an evidence-driven stream that enforces strict grounding in the image. These two streams are adaptively fused using a symmetric KL-based contrast-based gate, which suppresses tokens favored by language priors but unsupported by visual evidence, while preserving them when both distributions agree. We evaluate IECD2 on multiple datasets spanning various generative vision-language reasoning tasks such as captioning and visual question answering, including POPE, MME, VQAv2, AMBER, MS-COCO, and LLaVA-Bench. IECD2 demonstrates consistent improvements in task accuracy and reasoning performance, alongside a substantial reduction in hallucination across all evaluation metrics compared to state-of-the-art decoding approaches.

*   **研究背景**：视觉语言模型（VLMs）虽然在指令遵循和开放式推理上表现出色，但极易产生“幻觉”（Hallucination），即生成与图像内容不符的描述。
*   **核心问题**：先前的研究发现，指令提示（Instruction Prompting）会放大语言模型的先验知识（Language Priors）。当视觉信号模糊时，模型倾向于根据语言习惯“脑补”内容，而非依据视觉证据。
*   **研究动机**：如何在不进行重新训练的前提下，在解码阶段动态平衡“回复的丰富性（指令驱动）”与“视觉的忠实度（证据驱动）”。

### 2. 方法论
IECD² 提出了一种**无需训练（Training-free）**的双流对比解码框架：
*   **核心思想**：在每个解码步，同时维护两个并行的 Token 概率分布流，通过对比两者的差异来识别并抑制幻觉。
*   **关键技术细节**：
    *   **指令流（Instruction Stream）**：使用标准指令提示词，旨在生成流畅、信息丰富的响应，但包含较强语言先验。
    *   **证据流（Evidence Stream）**：使用严格限制的提示词（如“请严格根据图中可见内容回答”），旨在生成保守但高度忠实于视觉的响应。
    *   **自适应对比门控（Contrastive Gating）**：
        *   使用**对称 KL 散度（Symmetric KL Divergence）**来衡量两个流之间的分歧程度。
        *   **逻辑**：如果两个流分歧很大，说明指令流可能正在受语言先验驱动产生幻觉，此时门控机制会降低指令流的权重，强制模型转向证据流；如果两者一致，则保留指令流的丰富表达。
    *   **融合公式**：通过几何平均融合（Geometric-mean fusion）在对数空间对两个分布进行加权组合。

### 6. 主要结论与发现
*   **显著降低幻觉**：在 MS-COCO 任务中，IECD² 将幻觉指标 CHAIR 降低了 **85% 以上**；在 AMBER 任务中，认知幻觉（Cog）降低了约 **98%**。

*   **平衡性**：证明了单纯依靠“证据流”会导致输出过于保守（字数少、信息量低），而 IECD² 的双流融合能有效兼顾“说得对”和“说得好”。

### 7. 优点
*   **即插即用**：不需要对模型进行微调或参数更新，适用于任何已有的指令微调 VLM。
*   **动态自适应**：相比于固定权重的对比解码，基于 KL 散度的门控能根据 Token 的不确定性实时调整干预强度。
*   **鲁棒性强**：在对抗性（Adversarial）和流行性（Popular）偏差的数据集上表现出比同类方法更好的稳定性。

### 8. 不足与局限
*   **计算开销**：尽管有 KV Cache，但双流解码本质上需要两次前向传播（或在 Batch 中并行），推理成本仍高于标准解码。
*   **提示词敏感性**：证据流的效果高度依赖于“证据提示词”的设计，不同模型可能需要微调提示词以达到最佳效果。
*   **模型规模限制**：实验主要基于 7B 规模的模型，尚未在更大规模（如 70B）或更小规模（端侧模型）上验证其泛化性。
