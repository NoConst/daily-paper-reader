## 摘要
大语言模型（LLM）智能体需要长期用户记忆来实现一致的个性化，但有限的上下文窗口阻碍了在长时交互中追踪演化的偏好。现有的记忆系统主要依赖于静态的手工更新规则；虽然基于强化学习（RL）的智能体可以学习记忆更新，但稀疏的结果奖励提供了微弱的监督，导致长程优化不稳定。借鉴记忆图式理论以及前额叶区域与海马体区域之间的功能划分，我们提出了 MemCoE，这是一个受认知启发的两阶段优化框架，用于学习记忆应如何组织以及更新哪些信息。在第一阶段，我们提出了记忆指南归纳（Memory Guideline Induction），通过将对比反馈解释为文本梯度来优化全局指南；在第二阶段，指南对齐的记忆策略优化（Guideline-Aligned Memory Policy Optimization）利用归纳出的指南来定义结构化的过程奖励，并进行多轮强化学习，以学习遵循指南的记忆演化策略。我们在三个个性化记忆基准上进行了评估，涵盖了显式/隐式偏好以及不同的规模和噪声，并观察到相比强基线模型有持续的改进，且具有良好的鲁棒性、可迁移性和效率。

## Abstract
Large language model (LLM) agents require long-term user memory for consistent personalization, but limited context windows hinder tracking evolving preferences over long interactions. Existing memory systems mainly rely on static, hand-crafted update rules; although reinforcement learning (RL)-based agents learn memory updates, sparse outcome rewards provide weak supervision, resulting in unstable long-horizon optimization. Drawing on memory schema theory and the functional division between prefrontal regions and hippocampus regions, we introduce MemCoE, a cognition-inspired two-stage optimization framework that learns how memory should be organized and what information to update. In the first stage, we propose Memory Guideline Induction to optimize a global guideline via contrastive feedback interpreted as textual gradients; in the second stage, Guideline-Aligned Memory Policy Optimization uses the induced guideline to define structured process rewards and performs multi-turn RL to learn a guideline-following memory evolution policy. We evaluate on three personalization memory benchmarks, covering explicit/implicit preference and different sizes and noise, and observe consistent improvements over strong baselines with favorable robustness, transferability, and efficiency.

#### 1. 核心问题与整体含义（研究动机和背景）
大语言模型（LLM）智能体在提供个性化服务时，面临**长程交互中用户偏好动态演化**的挑战。由于 LLM 的上下文窗口有限，无法直接处理全部历史记录。现有的记忆系统主要存在两类问题：
*   **静态规则局限性：** 依赖手工设计的提取和更新规则，难以适应多样化的用户行为和领域。
*   **强化学习（RL）优化不稳定：** 虽然有研究尝试用 RL 学习记忆更新，但通常仅依赖最终答案的“稀疏奖励”，导致在长时交互中探索空间过大、训练不稳定且效率低下。
**研究动机：** 借鉴认知心理学的“记忆图式理论”，将记忆过程解耦为“前额叶区域（定义组织模式/图式）”与“海马体区域（填充具体情节细节）”的功能分工。

#### 2. 方法论：核心思想与技术细节
论文提出了 **MemCoE** 框架，通过两个阶段优化“如何记忆”和“记忆什么”：

*   **第一阶段：记忆指南归纳 (Memory Guideline Induction, MGI)**
    *   **核心思想：** 学习“如何组织记忆”。将记忆更新指令（Prompt）视为可优化的全局文本参数。
    *   **关键技术：** 
        *   **对比反馈作为文本梯度：** 比较正确和错误的记忆演化轨迹，利用 LLM 生成自然语言形式的“文本梯度”，指出当前指南的不足。
        *   **批次梯度聚合：** 在 mini-batch 上聚合这些反馈，消除随机噪声，诱导出通用的、跨领域的记忆组织准则。
*   **第二阶段：指南对齐的记忆策略优化 (Guideline-Aligned Memory Policy Optimization, GMPO)**
    *   **核心思想：** 学习“更新哪些内容”。在固定第一阶段诱导出的指南基础上，训练具体的记忆更新策略。
    *   **关键技术：**
        *   **指南对齐奖励 (Guideline-aligned rewards)：** 除了最终答案的正确性奖励外，引入“过程奖励”，即评估记忆更新是否严格遵循了第一阶段定义的结构和规则。
        *   **多轮强化学习：** 使用 **GRPO (Group Relative Policy Optimization)** 算法，在多轮对话轨迹上端到端地优化记忆演化策略。

#### 5. 实验数量与充分性
*   **实验规模：** 涵盖了 3 个基准测试、8 种不同的评估设置（不同长度、不同偏好类型、不同噪声等级）。
*   **消融实验：** 详细验证了对比反馈（CF）、指南奖励（GR）以及 MGI 和 GMPO 两个阶段的必要性。

#### 6. 主要结论与发现
*   **性能领先：** MemCoE 在所有基准测试中均显著优于强基线模型（如 MemAgent 和 A-Mem）。
*   **稳定性与效率：** 通过引入过程奖励，MemCoE 解决了纯 RL 训练不稳定的问题，且在推理效率（性能-时间平衡）上表现优异。
*   **强迁移性：** 在 Qwen 上诱导出的记忆指南，直接应用到 GPT-4o 或 Gemini 上也能显著提升性能，证明了“记忆准则”具有模型无关的通用性。
*   **抗噪性：** 在高噪声环境下，MemCoE 依然能有效过滤无关信息，保持对核心偏好的记忆。

#### 7. 优点
*   **认知启发：** 巧妙地将心理学理论转化为工程上的两阶段优化，解耦了组织逻辑与具体内容。
*   **解决稀疏奖励问题：** 通过诱导指南并将其转化为过程奖励，为 RL 提供了更细粒度的监督信号。
*   **通用性强：** 诱导出的文本指南具有良好的可解释性和跨模型迁移能力。

#### 8. 不足与局限
*   **评分器依赖：** 第二阶段的奖励计算依赖于 LLM 作为评分器，评分器的可靠性可能影响训练效果。
*   **超参数敏感：** 实验表明每轮演化的 Token 预算和演化轮数对结果有显著影响，实际部署时需要仔细调优。
*   **错误累积风险：** 在极长历史的多次演化过程中，微小的更新错误可能会随时间累积，导致意外的遗忘或记忆漂移。
