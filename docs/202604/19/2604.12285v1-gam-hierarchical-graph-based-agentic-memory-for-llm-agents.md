## 摘要
为了维持连贯的长期交互，大语言模型（LLM）智能体必须在获取新信息与保留先验知识之间寻求平衡。目前的统一流式记忆系统虽然便于上下文更新，但容易受到瞬时噪声的干扰。相反，离散的结构化记忆架构虽然能提供稳健的知识保留，但往往难以适应不断演变的叙事。为了解决这一问题，我们提出了 GAM，一种基于分层图的智能体记忆框架。该框架通过显式地将记忆编码与巩固解耦，有效解决了快速上下文感知与稳定知识保留之间的冲突。通过将正在进行的对话隔离在事件推进图中，并仅在发生语义偏移时将其整合到主题关联网络中，我们的方法在保持长期一致性的同时最大限度地减少了干扰。此外，我们引入了一种图引导的多因素检索策略，以提高上下文的精准度。在 LoCoMo 和 LongDialQA 上的实验表明，我们的方法在推理准确性和效率方面均持续优于现有的最先进基准模型。

## Abstract
To sustain coherent long-term interactions, Large Language Model (LLM) agents must navigate the tension between acquiring new information and retaining prior knowledge. Current unified stream-based memory systems facilitate context updates but remain vulnerable to interference from transient noise. Conversely, discrete structured memory architectures provide robust knowledge retention but often struggle to adapt to evolving narratives. To address this, we propose GAM, a hierarchical Graph-based Agentic Memory framework that explicitly decouples memory encoding from consolidation to effectively resolve the conflict between rapid context perception and stable knowledge retention. By isolating ongoing dialogue in an event progression graph and integrating it into a topic associative network only upon semantic shifts, our approach minimizes interference while preserving long-term consistency. Additionally, we introduce a graph-guided, multi-factor retrieval strategy to enhance context precision. Experiments on LoCoMo and LongDialQA indicate that our method consistently outperforms state-of-the-art baselines in both reasoning accuracy and efficiency.

#### 1. 核心问题与整体含义（研究动机和背景）
大语言模型（LLM）智能体在处理长期交互时面临一个核心矛盾：**快速感知新信息**与**稳定保留长期知识**之间的冲突。
*   **现有系统的缺陷：**
    *   **统一流式记忆系统（如 MemGPT, Mem0）：** 将新旧信息混杂在单一流中，容易导致“记忆污染”，表现为**记忆丢失**（旧节点因连接弱被遗忘）和**语义漂移**（不同主题被错误合并）。
    *   **离散结构化记忆（如 GraphRAG）：** 结构过于僵化，难以实时捕捉对话的动态演变，导致叙事表示碎片化。
*   **研究动机：** 借鉴人类睡眠驱动的记忆巩固机制，提出一种能解耦“编码”与“巩固”过程的记忆框架，以实现鲁棒的长期一致性。

#### 2. 方法论
GAM 提出了一个**分层图记忆架构**，通过**语义事件触发机制**将记忆生命周期分为两个阶段：

*   **核心架构：**
    *   **底层：事件推进图（Event Progression Graphs, $G_{event}$）：** 局部图结构，负责隔离并存储当前对话的原子单元（如用户话语、系统响应），保护全局存储不受瞬时噪声干扰。
    *   **顶层：主题关联网络（Topic Associative Network, $G_{topic}$）：** 全局图结构，存储高层语义集群和抽象主题，记录主题间的深层相关性。
*   **关键技术细节：**
    *   **状态切换机制：** 系统在“情节缓冲状态”和“语义巩固状态”间切换。通过 LLM 充当神经代理，检测语义边界（$b_t$）。当检测到主题偏移时，触发巩固。
    *   **语义巩固：** 将缓冲区的局部图总结为具有“双粒度”表示（摘要 + 原始文本）的主题节点，并利用“粗到细”的策略将其整合进全局网络。
    *   **图引导多因子检索：** 采用“自顶向下”的遍历策略。先定位语义锚点，再通过跨层链接下钻到具体情节，最后结合**时间（Temporal）、置信度（Confidence）、角色（Role）**三个因子进行重排序。

#### 6. 主要结论与发现
*   **性能领先：** GAM 在 LoCoMo 和 LongDialQA 上均显著优于基线。例如，在 Qwen2.5-7B 上，其时间推理 F1 分数比 Mem0 高出 18%。

#### 7. 优点
*   **架构创新：** 显式解耦编码与巩固过程，解决了流式记忆系统长期存在的“记忆污染”问题。
*   **双粒度存储：** 既保留了高层摘要用于快速检索，又保留了原始文本用于精确回溯，平衡了抽象与细节。
*   **检索精准：** 引入角色和时间等多因子调制，使智能体在处理复杂社交关系和时间线时更具优势。
*   **即插即用：** 无需训练，可适配多种开源和闭源 LLM。

*   **模态限制：** 目前仅支持文本模态，无法直接处理视觉或音频等非语言信号（尽管作者在附录中做了代理多模态实验）。

*   **计算开销：** 虽然 Token 效率高，但维护复杂的图结构和频繁调用 LLM 进行边界检测/关系评分仍存在一定的计算延迟（Latency 略高于 Mem0）。
