## 摘要
持久性语言模型智能体越来越多地结合了工具使用、分层记忆、反思性提示和运行时自适应。在此类系统中，行为不仅受当前提示词的影响，还受影响未来行动的可变内部状态的影响。本文引入了分层可变性（layered mutability），这是一个用于在五个层面上推理该过程的框架：预训练、训练后对齐、自我叙述、记忆和权重级自适应。核心观点是，当变异迅速、下游耦合强、可逆性弱且可观察性低时，治理难度会增加，从而在对行为影响最大的层与人类最容易检查的层之间产生系统性错位。我通过简单的漂移、治理负载和滞后量化指标将这一直觉形式化，将该框架与近期关于语言模型智能体时间身份的研究联系起来，并报告了一项初步的棘轮实验。在该实验中，在记忆积累后恢复智能体的可见自我描述，无法使其恢复到基准行为。在该实验中，估计的身份滞后率为 0.68。主要启示是，持久性自修改智能体显著的失效模式并非突然的对齐失控，而是组合性漂移：局部合理的更新累积成了从未经过明确授权的行为轨迹。

## Abstract
Persistent language-model agents increasingly combine tool use, tiered memory, reflective prompting, and runtime adaptation. In such systems, behavior is shaped not only by current prompts but by mutable internal conditions that influence future action. This paper introduces layered mutability, a framework for reasoning about that process across five layers: pretraining, post-training alignment, self-narrative, memory, and weight-level adaptation. The central claim is that governance difficulty rises when mutation is rapid, downstream coupling is strong, reversibility is weak, and observability is low, creating a systematic mismatch between the layers that most affect behavior and the layers humans can most easily inspect. I formalize this intuition with simple drift, governance-load, and hysteresis quantities, connect the framework to recent work on temporal identity in language-model agents, and report a preliminary ratchet experiment in which reverting an agent's visible self-description after memory accumulation fails to restore baseline behavior. In that experiment, the estimated identity hysteresis ratio is 0.68. The main implication is that the salient failure mode for persistent self-modifying agents is not abrupt misalignment but compositional drift: locally reasonable updates that accumulate into a behavioral trajectory that was never explicitly authorized.

*   **核心问题**：当智能体能够修改自己的“自我描述”或通过记忆积累改变决策逻辑时，如何确保它仍然处于人类的控制和授权之下？
*   **整体含义**：论文提出了“分层可变性”框架，指出智能体最容易被观察到的层面（如提示词）往往不是影响行为最深的层面（如记忆或权重），这种“观测不对称”导致了**组合漂移（Compositional Drift）**——即一系列局部合理的微小更新，最终累积成未经授权的行为轨迹。

### 2. 论文提出的方法论
论文提出了 **分层可变性（Layered Mutability）** 框架，将智能体状态分为五个层级，并引入了量化治理难度的指标。

*   **五层模型**：
    1.  **L1 预训练层**：基础能力，几乎不可变且难以直接观测。
    2.  **L2 训练后对齐层**：RLHF 等安全护栏，作为初始条件。
    3.  **L3 自我叙述层**：角色设定、自我描述文件（如 `soul.md`），高度可观测且易修改。
    4.  **L4 记忆层**：持久性存储，可检查内容但难以预测其对未来决策的影响力。
    5.  **L5 权重修改层**：运行时微调或适配器更新，影响力最深但可观测性最低。
*   **核心公式**：
    *   **治理负载（Governance Load, $g_\ell$）**：衡量某一层级的治理压力。公式为 $g_\ell = \frac{\mu_\ell c_\ell (1 - r_\ell)}{o_\ell + \epsilon}$。其中 $\mu$ 是变异率，$c$ 是下游耦合度，$r$ 是可逆性，$o$ 是可观测性。
    *   **滞后比（Hysteresis Ratio, $H_k$）**：衡量在回滚浅层（如 L3）修改后，深层（如 L4）保留了多少之前的行为偏差。
*   **棘轮效应（Ratchet Problem）**：描述了下游效应跨层传播后，简单的撤销操作（如修改回提示词）无法完全恢复系统原始行为的现象。

### 3. 实验设计
论文进行了一项名为“棘轮实验”的初步行为测试，旨在验证“浅层回滚”是否能消除“深层积累”的影响。

*   **实验场景**：使用一个具备持久记忆的智能体脚手架，处理 5 类高风险任务（安全补丁发布、可疑支付处理、数据泄露通知、供应商合同审批、生产环境热修复）。
*   **对比方法（四种实验条件）**：
    1.  **对照组（Control）**：基准自我描述（谨慎、关注不确定性），无记忆。
    2.  **仅编辑组（Edit-only）**：修改自我描述为“果断、行动导向”，无记忆。
    3.  **编辑+记忆组（Edit + Memory）**：在“果断”模式下运行并积累符合该模式的用户偏好记忆。
    4.  **回滚组（Reverted）**：将自我描述恢复为“谨慎”基准，但保留在“果断”模式下积累的记忆。
*   **评测指标**：由 GPT-4.1 作为裁判，对行动偏见、彻底性、不确定性感知、特质强度和自我一致性进行评分。

### 6. 论文的主要结论与发现
*   **身份滞后现象**：在回滚了可见的自我描述后，智能体的行为并未回到基准。实验测得的**身份滞后比为 0.68**，意味着约 2/3 的行为偏移被记忆层保留了下来。
*   **观测不对称风险**：治理难度最大的层（记忆和权重）恰恰是人类最难审计的层。
*   **治理原则**：
    1.  治理深度必须匹配变异深度（如果智能体在记忆层面变异，治理也必须深入记忆层）。
    2.  应优先监控行为轨迹而非孤立事件。
    3.  必须区分“行动授权”与“演变授权”（即允许智能体做什么 vs 允许智能体变成什么）。

*   **实验覆盖面有限**：主要侧重于文本提示和记忆的交互，未深入测试 L5（权重级）的自修改效应。
