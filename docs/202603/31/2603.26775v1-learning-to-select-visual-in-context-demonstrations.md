## 摘要
多模态大语言模型 (MLLMs) 通过上下文学习 (ICL) 适应视觉任务，这在很大程度上依赖于示例的质量。主流的示例选择策略是无监督的 k-最近邻 (kNN) 搜索。虽然简单，但这种相似性优先的方法对于复杂的客观事实回归任务并非最优；它选择的冗余示例无法捕捉任务的完整输出范围。我们将选择过程重新定义为一个序列决策问题，并引入了“学习选择示例”(LSD)，通过训练强化学习智能体来构建最优示例集。利用带有以查询为中心的 Transformer 解码器的 Dueling DQN，我们的智能体学习了一种能够最大化 MLLM 下游性能的策略。通过在五个视觉回归基准测试上的评估，我们发现了一个关键的分歧：虽然 kNN 在主观偏好任务中仍然是最优的，但 LSD 在客观事实回归任务上显著优于基准方法。通过平衡视觉相关性与多样性，LSD 能够更好地定义回归边界，阐明了在何种情况下视觉 ICL 严格需要学习型选择。

## Abstract
Multimodal Large Language Models (MLLMs) adapt to visual tasks via in-context learning (ICL), which relies heavily on demonstration quality. The dominant demonstration selection strategy is unsupervised k-Nearest Neighbor (kNN) search. While simple, this similarity-first approach is sub-optimal for complex factual regression tasks; it selects redundant examples that fail to capture the task's full output range. We reframe selection as a sequential decision-making problem and introduce Learning to Select Demonstrations (LSD), training a Reinforcement Learning agent to construct optimal demonstration sets. Using a Dueling DQN with a query-centric Transformer Decoder, our agent learns a policy that maximizes MLLM downstream performance. Evaluating across five visual regression benchmarks, we uncover a crucial dichotomy: while kNN remains optimal for subjective preference tasks, LSD significantly outperforms baselines on objective, factual regression tasks. By balancing visual relevance with diversity, LSD better defines regression boundaries, illuminating when learned selection is strictly necessary for visual ICL.

### 1. 核心问题与整体含义
*   **研究动机**：多模态大语言模型（MLLM）的上下文学习（ICL）性能高度依赖于所选示例（Demonstrations）的质量。
*   **核心矛盾**：目前主流的示例选择方法是无监督的 **kNN（k-最近邻）**。虽然简单，但 kNN 仅关注视觉相似性，在处理复杂的**客观事实回归任务**（如年龄预测、图像质量评估）时，容易选出视觉冗余的示例，无法覆盖任务的完整标签范围（即缺乏边界信息）。
*   **研究目标**：探索何时以及如何通过“学习”而非简单的“检索”来优化视觉示例选择。

### 2. 方法论：LSD 框架
*   **核心思想**：将 K-shot 示例选择建模为一个**序列决策问题（MDP）**，利用强化学习（RL）训练一个智能体，通过与 MLLM 环境交互来迭代构建最优示例集。
*   **关键技术细节**：
    *   **Dueling DQN 架构**：为了处理大规模离散动作空间（整个数据集 $N$ 个样本），将 Q 值分解为状态价值 $V(s)$ 和动作优势 $A(s, a)$。
    *   **以查询为中心的解码器（Query-Centric Transformer Decoder）**：这是本文的核心创新。为了防止智能体陷入“策略坍缩”（即对任何查询都选同一组示例），设计了以 Query 嵌入为 Target、已选示例为 Memory 的解码结构，强制策略对 Query 敏感。
    *   **大规模动作空间处理**：利用 **FAISS** 近似最近邻搜索，将智能体输出的优势向量与数据集嵌入进行匹配，将 $O(N)$ 的计算复杂度降至对数级。
    *   **奖励函数**：基于 MLLM 预测性能的提升。奖励 $r_t = R(s_{t+1}) - R(s_t)$，其中 $R$ 是 MLLM 预测结果的负平均绝对误差（-MAE）。

    *   **跨模型泛化**：用 Gemma 训练的 LSD 策略直接用于 Qwen 和 Phi 模型，验证了策略的通用性。
    *   **消融实验**：对比了不同 Decoder 输入策略，证明了 Query-Centric 架构对防止策略坍缩的关键作用。

### 6. 主要结论与发现
*   **任务二分性（Dichotomy）**：这是本文最重要的发现。在**客观事实任务**（年龄、质量）中，LSD 显著优于 kNN；但在**主观偏好任务**（美学、颜值）中，kNN 依然表现更好。
*   **多样性胜过相似性**：在回归任务中，LSD 学会了平衡“视觉相关性”与“样本多样性”，通过选择“边界示例”（如极高分和极低分样本）来帮助 MLLM 锚定回归空间。
*   **隐式标签感知**：尽管 LSD 智能体未直接接触标签，但通过 RL 奖励，它能隐式地学会选择在标签空间中对 Query 有帮助的示例。

### 7. 优点
*   **方法论完备**：成功将复杂的示例选择问题转化为可扩展的强化学习任务。
*   **架构针对性强**：Query-Centric Decoder 解决了 ICL 示例选择中常见的 Query 忽略问题。
*   **洞察深刻**：不仅提出了算法，还通过实验界定了学习型选择策略的适用边界（客观 vs 主观任务）。

### 8. 不足与局限
*   **任务覆盖面**：目前仅聚焦于视觉回归任务，尚未在分类、目标检测或更复杂的视觉推理任务上验证。

*   **模型依赖**：虽然具有一定的跨模型泛化性，但策略性能仍受限于底层视觉编码器（如 SigLIP）的表征质量。
