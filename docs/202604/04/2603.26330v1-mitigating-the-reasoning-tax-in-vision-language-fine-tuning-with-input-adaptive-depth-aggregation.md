## 摘要
在视觉指令数据上进行有监督微调（SFT）通常会提升视觉语言模型（VLM）的感知能力，但同时会降低其推理性能，从而在训练后阶段产生持续的“推理代价”。我们研究了这种性能下降是否与深度表征访问受阻有关，并发现即使是固定的跨深度聚合也能显著恢复推理能力，这表明保持跨深度访问是 VLM 微调中一个重要的缺失因素。基于这一观察，我们提出了输入自适应深度聚合（IADA），这是一种轻量级机制，通过低秩瓶颈实现了输入自适应、模态感知的跨深度检索，且参数化效率极高。在 Qwen3-VL-2B 模型上，相比于仅使用 LoRA 的微调，IADA 仅增加了 0.14M 额外参数，就将平均推理分数提升了 9.5 分，平均感知分数提升了 3.3 分，且在参数高效的低秩设置下增益最为显著。

## Abstract
Supervised fine-tuning (SFT) on visual instruction data often improves perceptual capabilities in vision-language models (VLMs) while degrading reasoning performance, creating a persistent reasoning tax during post-training. We investigate whether this degradation is related to disrupted access to depth-wise representations, and find that even fixed cross-depth aggregation substantially restores reasoning, suggesting that preserved cross-depth access is an important missing factor in VLM fine-tuning. Building on this observation, we propose Input-Adaptive Depth Aggregation (IADA), a lightweight mechanism that makes cross-depth retrieval input-adaptive, modality-aware, and efficiently parameterized through a low-rank bottleneck. On Qwen3-VL-2B, IADA improves the average reasoning score by 9.5 points and the average perception score by $3.3$ points over LoRA-only fine-tuning with only 0.14M additional parameters, with the strongest gains appearing in parameter-efficient low-rank settings.

*   **研究背景**：视觉语言模型（VLM）在经过有监督微调（SFT）后，虽然感知能力（如OCR、图像识别）有所提升，但往往会出现推理能力（如数学、科学推理）下降的现象，这种现象被称为“推理税”（Reasoning Tax）。
*   **核心假设**：作者假设 VLM 在预训练中形成了深度特征层级（浅层负责感知，深层负责抽象推理）。统一的微调过程破坏了模型对这些分布在不同深度表征的有效访问。
*   **研究目的**：通过引入跨深度的特征聚合机制，恢复模型对推理相关表征的访问，从而在不牺牲感知能力的前提下缓解推理性能的退化。

### 2. 方法论：输入自适应深度聚合 (IADA)
*   **核心思想**：在 Transformer 块的边界插入轻量级模块，根据当前输入动态地从之前的层级中检索并聚合特征。
*   **关键技术细节**：
    1.  **输入自适应 Query 生成**：对当前隐藏状态进行全局平均池化（MeanPool），通过一个**低秩瓶颈结构**（$W_{up} \cdot W_{down}$）生成查询向量（Query）。这使得路由决策取决于输入内容。
    2.  **模态感知路由**：利用模态掩码（Modality Mask）将视觉和文本 Token 分开，分别为两种模态生成独立的聚合查询，以适应不同模态在深度方向上的处理模式。
    3.  **跨深度交叉注意力**：使用生成的 Query 从缓存的历史块边界隐藏状态中检索信息。
    4.  **低秩门控注入**：将检索到的残差信息通过一个可学习的门控（Gated Residual）加回到当前隐藏状态中。
*   **算法流程**：输入状态 -> 模态分离 -> 池化与低秩映射生成 Query -> 跨层注意力检索历史状态 -> 门控残差融合 -> 更新状态。

*   **结构创新**：将跨层路由视为独立于层内适配的全新设计维度。
*   **即插即用**：作为 PEFT 的补充，可以轻松集成到现有的 Transformer 架构中。
*   **诊断价值**：通过固定聚合实验，为理解 VLM 微调中的性能退化提供了新的机械论视角。
