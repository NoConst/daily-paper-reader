## 摘要
稠密向量检索是检索增强生成（RAG）的实际支柱，但相似度搜索可能面临精度限制。相反，利用大语言模型（LLM）重排序的基于效用的方法通常能获得卓越的性能，但计算成本极高，且容易受到困惑度（perplexity）估计中固有噪声的影响。我们提出了效用对齐嵌入（Utility-Aligned Embeddings, UAE），这是一个旨在将这些优势融合为一种实用、高性能检索方法的框架。我们将检索建模为一个分布匹配问题，通过使用效用调制的 InfoNCE（Utility-Modulated InfoNCE）目标函数，训练双编码器（bi-encoder）来模仿源自困惑度降低的效用分布。该方法直接将分级效用信号注入嵌入空间，而无需在测试时进行 LLM 推理。在 QASPER 基准测试中，相比于强大的语义基准模型 BGE-Base，UAE 将检索 Recall@1 提升了 30.59%，MAP 提升了 30.16%，Token F1 提升了 17.3%。至关重要的是，UAE 比高效的 LLM 重排序方法快 180 倍以上，同时保持了竞争力的性能，这表明将检索与生成效用对齐可以大规模地提供可靠的上下文。

## Abstract
Dense vector retrieval is the practical backbone of Retrieval- Augmented Generation (RAG), but similarity search can suffer from precision limitations. Conversely, utility-based approaches leveraging LLM re-ranking often achieve superior performance but are computationally prohibitive and prone to noise inherent in perplexity estimation. We propose Utility-Aligned Embeddings (UAE), a framework designed to merge these advantages into a practical, high-performance retrieval method. We formulate retrieval as a distribution matching problem, training a bi-encoder to imitate a utility distribution derived from perplexity reduction using a Utility-Modulated InfoNCE objective. This approach injects graded utility signals directly into the embedding space without requiring test-time LLM inference. On the QASPER benchmark, UAE improves retrieval Recall@1 by 30.59%, MAP by 30.16% and Token F1 by 17.3% over the strong semantic baseline BGE-Base. Crucially, UAE is over 180x faster than the efficient LLM re-ranking methods preserving competitive performance, demonstrating that aligning retrieval with generative utility yields reliable contexts at scale.

### 1. 核心问题与研究动机
*   **核心问题**：传统的稠密检索器（Bi-encoders）依赖语义相似度，但高相似度的文档不一定对大模型（LLM）生成正确答案最有帮助（即“效用”不高），甚至可能成为干扰项。
*   **研究动机**：虽然基于 LLM 的重排序（Reranking）效果好，但计算开销巨大（延迟高），且直接利用 LLM 的困惑度（Perplexity）作为训练信号存在噪声大、分布不均、计算昂贵等问题。
*   **目标**：在保持双编码器极速推理优势的同时，将 LLM 的生成偏好（效用信号）注入到嵌入空间中。

### 2. 方法论
UAE 采用两阶段蒸馏框架，将检索建模为分布匹配问题：
*   **阶段 A：参数化效用近似（奖励模型训练）**：
    *   首先定义“效用”：通过 LLM 在有无该背景文档时生成正确答案的困惑度下降程度来衡量。
    *   为了克服原始效用分数的高方差和长尾分布，训练一个基于 DeBERTa 的**奖励模型（Reward Model）**。该模型不直接回归分数，而是通过**边际排名损失（Margin-based Ranking Loss）**学习文档间的相对效用顺序。
*   **阶段 B：奖励引导的嵌入训练（分布匹配）**：
    *   使用训练好的奖励模型作为“教师”，通过 **Utility-Modulated InfoNCE** 目标函数指导双编码器（学生）。
    *   **核心公式**：最小化教师分布（基于奖励模型分数）与学生分布（基于向量点积相似度）之间的 **KL 散度**。这使得检索器不仅学习“是或否”，还学习文档效用的分级结构。
    *   **效用门控负采样（Utility-Gated Mining）**：专门挖掘那些语义相似但效用极低的“语义干扰项”作为负样本，强制模型区分它们。
*   **阶段 C：推理**：
    *   训练完成后，仅保留双编码器。推理过程与标准向量检索完全一致，支持 ANN（近似最近邻）搜索。
