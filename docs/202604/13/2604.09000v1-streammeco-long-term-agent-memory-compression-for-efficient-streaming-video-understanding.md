## 摘要
视觉智能体记忆在流式视频理解中展现出了显著的有效性。然而，为视频存储此类记忆会产生巨大的内存开销，导致存储和计算成本高昂。为了解决这一问题，我们提出了 StreamMeCo，一个高效的流式智能体记忆压缩框架。具体而言，基于记忆图的连通性，StreamMeCo 为孤立节点引入了无边最大最小采样（edge-free minmax sampling），并为连接节点引入了边感知权重剪枝（edge-aware weight pruning），在保持准确性的同时剔除冗余记忆节点。此外，我们引入了一种时间衰减记忆检索机制，以进一步消除记忆压缩带来的性能下降。在三个具有挑战性的基准数据集（M3-Bench-robot、M3-Bench-web 和 Video-MME-Long）上的广泛实验表明，在 70% 的记忆图压缩率下，StreamMeCo 实现了 1.87 倍的记忆检索加速，同时平均准确率提升了 1.0%。我们的代码可在 https://github.com/Celina-love-sweet/StreamMeCo 获取。

## Abstract
Vision agent memory has shown remarkable effectiveness in streaming video understanding. However, storing such memory for videos incurs substantial memory overhead, leading to high costs in both storage and computation. To address this issue, we propose StreamMeCo, an efficient Stream Agent Memory Compression framework. Specifically, based on the connectivity of the memory graph, StreamMeCo introduces edge-free minmax sampling for the isolated nodes and an edge-aware weight pruning for connected nodes, evicting the redundant memory nodes while maintaining the accuracy. In addition, we introduce a time-decay memory retrieval mechanism to further eliminate the performance degradation caused by memory compression. Extensive experiments on three challenging benchmark datasets (M3-Bench-robot, M3-Bench-web and Video-MME-Long) demonstrate that under 70% memory graph compression, StreamMeCo achieves a 1.87* speedup in memory retrieval while delivering an average accuracy improvement of 1.0%. Our code is available at https://github.com/Celina-love-sweet/StreamMeCo.

#### 1. 核心问题与整体含义
随着流式视频理解（如直播、实时监控、自动驾驶）需求的增长，现有的视觉智能体（Vision Agent）通常将视频信息转化为基于文本、人脸和声音节点的“记忆图”（Memory Graph）。然而，随着视频时间的延长，记忆节点数量剧增，导致存储开销巨大且检索延迟极高，严重阻碍了实时响应。本文的研究动机在于：如何在大幅压缩记忆图规模的同时，保持甚至提升视频理解的准确率与检索效率。

#### 2. 方法论
StreamMeCo 提出了一种无需训练（Training-free）的记忆压缩框架，核心思想是根据记忆图中节点的**连通性**采取差异化压缩策略，并结合**人类记忆衰减机制**进行检索。

*   **双分支记忆压缩：**
    *   **孤立节点压缩（EMsampling）：** 针对不涉及特定实体的孤立文本节点，先使用球面 K-Means 算法进行聚类，再在每个簇内采用“最大最小采样”策略（先选中心点，再迭代选择与已选集合距离最远的点），以保证采样节点的代表性和多样性。
    *   **连接节点压缩（EWpruning）：** 针对与人脸/声音节点相连的实体文本节点，计算“实体重要性”（基于边权重）和“嵌入相似度”（基于多样性），通过加权融合得分来剪枝冗余节点。
*   **时间衰减记忆检索（TMR）：**
    *   受人类遗忘曲线启发，将记忆按时间分段。
    *   计算查询语句与各时间段记忆的相似度，并施加非线性时间衰减函数 $e^{-\lambda(t-j)}$。
    *   动态分配检索配额，优先检索近期关键信息和高相关性历史信息。
