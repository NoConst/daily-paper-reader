## 摘要
文本到图像生成技术发展迅速，但仍难以捕捉细微的用户偏好。现有方法通常依赖多模态大语言模型来推断用户偏好，但生成的提示词或潜码很少能忠实地反映这些偏好，导致个性化效果欠佳。我们提出了 Premier，一种用于个性化图像生成的创新偏好调制框架。Premier 将每个用户的偏好表示为可学习的嵌入，并引入了一个偏好适配器，将用户嵌入与文本提示词融合。为了实现准确且细粒度的偏好控制，融合后的偏好嵌入被进一步用于调制生成过程。为了增强个体偏好的区分度并提高输出与用户特定风格之间的对齐度，我们引入了分散损失，强制用户嵌入之间保持分离。当用户数据稀缺时，新用户被表示为训练期间学习到的现有偏好嵌入的线性组合，从而实现有效的泛化。实验表明，在相同的历史长度下，Premier 优于现有方法，在偏好对齐方面表现更强，并在文本一致性、ViPer 代理指标和专家评估中取得了卓越的性能。

## Abstract
Text-to-image generation has advanced rapidly, yet it still struggles to capture the nuanced user preferences. Existing approaches typically rely on multimodal large language models to infer user preferences, but the derived prompts or latent codes rarely reflect them faithfully, leading to suboptimal personalization. We present Premier, a novel preference modulation framework for personalized image generation. Premier represents each user's preference as a learnable embedding and introduces a preference adapter that fuses the user embedding with the text prompt. To enable accurate and fine-grained preference control, the fused preference embedding is further used to modulate the generative process. To enhance the distinctness of individual preference and improve alignment between outputs and user-specific styles, we incorporate a dispersion loss that enforces separation among user embeddings. When user data are scarce, new users are represented as linear combinations of existing preference embeddings learned during training, enabling effective generalization. Experiments show that Premier outperforms prior methods under the same history length, achieving stronger preference alignment and superior performance on text consistency, ViPer proxy metrics, and expert evaluations.

### 2. 方法论
Premier 的核心思想是将用户偏好参数化，并通过调制（Modulation）而非拼接的方式注入生成过程。
*   **可学习用户嵌入 ($e_u$)**：为每个用户分配一个可学习的向量，通过反向传播直接从用户的历史偏好图像中聚合信息。
*   **偏好适配器（Preference Adapter）**：
    *   利用交叉注意力机制，将用户嵌入作为 Key/Value，文本 Token 作为 Query。
    *   输出**偏好调制方向 ($\Delta$)**，分为“块共享（Block-shared）”和“块特定（Block-distinct）”两种，分别提供全局一致和层级特有的风格引导。
*   **色散损失（Dispersion Loss）**：
    *   为了防止适配器过度拟合文本而忽略用户特征，引入了基于 InfoNCE 的色散损失。
    *   **公式逻辑**：通过拉大不同用户在相同文本下的调制方向距离，增强个体偏好的区分度。
*   **新用户冷启动策略**：
    *   对于历史数据稀缺的新用户，将其嵌入表示为训练集中已有用户嵌入的**线性组合**。
    *   训练时仅优化组合系数，利用已有的稳定偏好空间实现快速泛化。

*   **数据集**：主要使用 **PrefBench**（包含 1000 名训练用户，每人 40-80 张偏好图；50 名测试用户）。补充实验使用了 **PIP-dataset**。

*   **实验规模**：涵盖了 1000+ 用户的训练和 50+ 用户的详尽测试，并进行了 40 名专家的 A/B Test 人工评估。
*   **消融实验**：针对块共享/块特定适配器、色散损失、提示词偏好调制（PPM）以及线性组合策略分别进行了消融，验证了各组件的必要性。
*   **充分性评价**：实验设计较为全面，不仅对比了客观指标，还通过可视化展示了在不同历史长度（2, 4, 8, 16, 32 张图）下的性能演变，客观地展示了线性组合策略在小样本下的优势。

### 6. 主要结论与发现
*   **性能领先**：Premier 在 ViPer 评分和人工偏好测试中显著优于 ViPer、DrUM 等现有 SOTA 方法。
*   **文本保真度**：相比 LoRA 容易导致模型“遗忘”指令（如数字、颜色指令失效），Premier 更好地保留了基础模型的文本遵循能力。
*   **泛化能力**：线性组合策略有效解决了新用户数据不足导致的过拟合问题，使模型在仅有 2-8 张参考图时仍能保持稳定。

### 7. 优点
*   **细粒度控制**：通过 Token 级别的调制，实现了比全局风格迁移更精准的偏好对齐。
*   **极高的效率**：极小的存储开销（KB 级别）使其在实际生产环境（支持海量用户个性化）中具有极强的应用潜力。
*   **创新的损失函数**：色散损失有效解决了偏好特征被文本特征淹没的难题。

### 8. 不足与局限
*   **训练集依赖**：实验表明，训练集用户规模（如从 1000 降至 100）会显著影响适配器的性能，这意味着该方法需要大规模的初始偏好数据来构建底座。

*   **应用限制**：目前主要基于 MM-DiT 架构（如 Flux），在传统的 UNet 架构上的迁移效果尚未验证。
