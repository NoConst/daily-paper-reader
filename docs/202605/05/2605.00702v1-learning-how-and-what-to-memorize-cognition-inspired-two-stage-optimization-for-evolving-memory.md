## 摘要
大语言模型（LLM）智能体需要长期用户记忆来实现一致的个性化，但有限的上下文窗口阻碍了在长期交互中追踪演化的偏好。现有的记忆系统主要依赖于静态的手工更新规则；虽然基于强化学习（RL）的智能体可以学习记忆更新，但稀疏的结果奖励提供了较弱的监督，导致长程优化不稳定。借鉴记忆图式理论以及前额叶区域与海马体区域之间的功能划分，我们提出了 MemCoE，这是一个受认知启发的两阶段优化框架，用于学习记忆应如何组织以及更新哪些信息。在第一阶段，我们提出了记忆指南归纳（Memory Guideline Induction），通过将对比反馈解释为文本梯度来优化全局指南；在第二阶段，指南对齐的记忆策略优化（Guideline-Aligned Memory Policy Optimization）利用归纳出的指南来定义结构化的过程奖励，并进行多轮强化学习，以学习遵循指南的记忆演化策略。我们在三个个性化记忆基准上进行了评估，涵盖了显式/隐式偏好以及不同的规模和噪声，并观察到相比强基线模型有持续的改进，且具有良好的鲁棒性、可迁移性和效率。

## Abstract
Large language model (LLM) agents require long-term user memory for consistent personalization, but limited context windows hinder tracking evolving preferences over long interactions. Existing memory systems mainly rely on static, hand-crafted update rules; although reinforcement learning (RL)-based agents learn memory updates, sparse outcome rewards provide weak supervision, resulting in unstable long-horizon optimization. Drawing on memory schema theory and the functional division between prefrontal regions and hippocampus regions, we introduce MemCoE, a cognition-inspired two-stage optimization framework that learns how memory should be organized and what information to update. In the first stage, we propose Memory Guideline Induction to optimize a global guideline via contrastive feedback interpreted as textual gradients; in the second stage, Guideline-Aligned Memory Policy Optimization uses the induced guideline to define structured process rewards and performs multi-turn RL to learn a guideline-following memory evolution policy. We evaluate on three personalization memory benchmarks, covering explicit/implicit preference and different sizes and noise, and observe consistent improvements over strong baselines with favorable robustness, transferability, and efficiency.

### 论文总结：Learning How and What to Memorize (MemCoE)

#### 1. 核心问题与整体含义（研究动机和背景）
大语言模型（LLM）在担任个性化助手时，面临着如何在有限的上下文窗口内持续追踪用户演化偏好的挑战。现有的记忆系统主要存在两类问题：
*   **静态规则局限性：** 许多系统依赖手工设计的提取和更新规则，难以适应复杂的交互场景和用户行为变化。
*   **强化学习（RL）优化不稳定：** 现有的基于 RL 的记忆更新方法通常只依赖最终任务的“结果奖励”（Outcome Rewards），这种奖励信号稀疏且延迟，导致在长程交互中优化困难，训练不稳定。

**研究动机：** 受认知心理学中“记忆图式理论（Memory Schema Theory）”的启发，人类大脑在前额叶（负责组织模式/图式）和海马体（负责填充具体细节）之间存在功能分工。论文据此提出了 **MemCoE** 框架，旨在将“如何组织记忆”与“记忆什么内容”进行解耦优化。

#### 2. 方法论：核心思想与关键技术
MemCoE 采用两阶段优化框架：

*   **第一阶段：记忆指南归纳 (Memory Guideline Induction, MGI)**
    *   **核心思想：** 模拟前额叶功能，学习全局的记忆组织准则（How）。
    *   **技术细节：** 将指令提示词（Prompt）视为可优化的文本参数。通过对比“正确”和“错误”的记忆更新轨迹，生成**对比反馈（Contrastive Feedback）**，并将其解释为**文本梯度（Textual Gradients）**。
    *   **批量聚合：** 在 Batch 级别聚合这些文本梯度，通过 LLM 迭代优化，最终诱导出一种领域通用的自然语言记忆指南。

*   **第二阶段：指南对齐的记忆策略优化 (Guideline-Aligned Memory Policy Optimization, GMPO)**
    *   **核心思想：** 模拟海马体功能，在指南约束下学习具体内容的更新策略（What）。
    *   **技术细节：** 使用第一阶段诱导出的指南来定义**结构化过程奖励（Process Rewards）**。
    *   **算法流程：** 采用 **GRPO（Group Relative Policy Optimization）** 算法进行多轮强化学习。奖励函数结合了“指南对齐得分”（是否遵循格式和逻辑）和“任务准确率得分”。通过这种方式，模型学习在遵循全局准则的同时，筛选对下游任务最有用的信息进行存储。

    2.  **PrefEval：** 测试模型在显式和隐式偏好下的多轮选择能力。
    3.  **PersonaBench：** 在含有不同程度噪声的异构用户语料库中进行个性化问答。
*   **对比方法（Baselines）：**

    *   **记忆库方法：** Mem0、A-Mem、LightMem（基于启发式规则或代理的记忆系统）。
    *   **RL 记忆方法：** MemAgent、Mem-α（基于强化学习的端到端更新方法）。

*   **训练细节：** 训练集包含从 PersonaMem 抽样的 300 个样本，采用多轮强化学习优化。

#### 5. 实验数量与充分性
论文进行了多维度的实验，具有较高的充分性和客观性：
*   **跨任务评估：** 在三个不同性质的基准测试（显式/隐式偏好、长文本、噪声环境）上进行了验证。
*   **消融实验：** 分别验证了对比反馈（CF）、指南奖励（GR）以及两个阶段（MGI 和 GMPO）的必要性。
*   **迁移性实验：** 验证了在一个模型上诱导出的“记忆指南”是否能直接应用于其他不同规模和系列的 LLM。
*   **效率分析：** 对比了记忆构建的时间成本与性能的平衡。
*   **鲁棒性分析：** 针对不同 Top-K 检索量、每轮 Token 预算和优化步数进行了参数敏感性分析。

#### 6. 主要结论与发现
*   **性能领先：** MemCoE 在所有基准测试中均显著优于强基线模型，尤其在处理长程演化偏好时表现突出。
*   **解耦的有效性：** 先学习“如何组织”再学习“更新什么”能显著降低 RL 的搜索空间，使训练更稳定。
*   **强迁移性：** 诱导出的自然语言指南具有模型无关性，可以跨 LLM 迁移并提升性能。
*   **抗噪性：** 在高噪声环境下，MemCoE 的性能下降比 RAG 和其他记忆系统更缓慢，证明其具备更好的信息筛选能力。

#### 7. 优点（亮点）
*   **认知启发：** 巧妙地将心理学中的图式理论转化为工程上的两阶段优化框架。
*   **过程监督：** 解决了传统 RL 记忆更新中奖励稀疏的问题，通过自动生成的指南提供密集的过程奖励。
*   **通用性：** 诱导出的指南是自然语言形式，易于理解且具备跨模型的可移植性。

#### 8. 不足与局限
*   **评分器依赖：** 第二阶段的奖励计算依赖于 LLM 作为评分器，评分器的可靠性会直接影响策略优化的效果。
*   **超参数敏感：** 实验表明模型对每轮演化的 Token 预算和演化轮数较为敏感，在实际部署中需要仔细调优。
*   **错误累积风险：** 在极长期的交互中，如果某一步记忆更新出现微小偏差，可能会随着演化轮数增加而产生累积效应，导致最终记忆失真。
