## 摘要
长期对话智能体需要能够捕捉事件间关系（而非仅仅是孤立事实）的记忆系统，以支持时间推理和多跳问答。目前的方法面临着一个基本的权衡：扁平化记忆效率高但无法建模关系结构，而基于图的记忆虽然能实现结构化推理，但构建成本高昂且脆弱。为了解决这些问题，我们提出了 StructMem，这是一种结构增强的分层记忆框架，它保留了事件级的绑定并诱导跨事件的连接。通过在时间上锚定双重视角并执行周期性的语义整合，StructMem 提升了在 LoCoMo 上的时间推理和多跳性能，同时与之前的记忆系统相比，显著减少了 Token 使用量、API 调用次数和运行时间，详见 https://github.com/zjunlp/LightMem 。

## Abstract
Long-term conversational agents need memory systems that capture relationships between events, not merely isolated facts, to support temporal reasoning and multi-hop question answering. Current approaches face a fundamental trade-off: flat memory is efficient but fails to model relational structure, while graph-based memory enables structured reasoning at the cost of expensive and fragile construction. To address these issues, we propose \textbf{StructMem}, a structure-enriched hierarchical memory framework that preserves event-level bindings and induces cross-event connections. By temporally anchoring dual perspectives and performing periodic semantic consolidation, StructMem improves temporal reasoning and multi-hop performance on \texttt{LoCoMo}, while substantially reducing token usage, API calls, and runtime compared to prior memory systems, see https://github.com/zjunlp/LightMem .

论文针对大语言模型（LLM）智能体在**长程对话（Long-Horizon Dialogue）**中的记忆挑战。现有的记忆系统主要分为两类：
*   **扁平记忆（Flat Memory）**：将事实或摘要存储为独立单元，效率高但缺乏事件间的关联，导致长历史检索退化为浅层的相似性匹配。
*   **图记忆（Graph Memory）**：通过实体-关系三元组构建结构，但构建成本极高（API调用多）、推理链条脆弱且容易产生幻觉累积。

**研究动机**：作者认为记忆的基本单元不应是孤立的事实或三元组，而应是**受时间锚定的关系事件（Temporally Grounded Relational Event）**。论文旨在提出一种既能保留结构化推理能力，又能保持高计算效率的记忆框架。

### 2. 方法论：StructMem 框架
StructMem 采用分层结构化设计，分为两个核心阶段：

*   **事件级绑定（Event-Level Binding）**：
    *   **双视角提取（Dual-Perspective Extraction）**：对每一条对话，同时从“事实视角”（内容）和“关系视角”（人际动态、因果影响、时间依赖）提取自然语言条目。
    *   **时间锚定（Temporal Anchoring）**：将所有提取的条目与其原始时间戳绑定，形成一个事件单元，避免了复杂的实体消解。
*   **跨事件整合（Cross-Event Consolidation）**：
    *   **语义事件连接**：当累积事件超过时间阈值时触发。系统利用当前缓存的事件作为查询，检索历史中语义相似的“种子条目”，并重构其完整的事件上下文。
    *   **合成式整合（Consolidation via Synthesis）**：LLM 将当前事件与检索到的相关历史事件进行批量合成，生成高层的关系假设（如因果链），从而在不丢失原始细节的前提下建立跨时间的结构化联系。

*   **实时性挑战**：虽然比图方法快，但周期性的整合操作在极高频交互场景下仍可能带来瞬时延迟。
