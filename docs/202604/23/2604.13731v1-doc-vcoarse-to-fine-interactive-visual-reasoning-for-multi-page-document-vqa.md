## 摘要
多页文档视觉问答（Multi-page Document Visual Question Answering）需要在长且视觉密集的文档中对语义、布局和视觉元素进行推理。现有的无 OCR（OCR-free）方法在容量和精度之间面临权衡：端到端模型在文档长度增加时扩展性较差，而基于视觉检索的流水线则显得脆弱且被动。我们提出了 Doc-$V^*$，这是一个无 OCR 的智能体（agentic）框架，它将多页 DocVQA 建模为序列化的证据聚合过程。Doc-$V^*$ 从缩略图概览开始，随后通过语义检索和有针对性的页面获取进行主动导航，并在结构化工作记忆中聚合证据以进行有据推理。通过专家轨迹的模仿学习进行训练，并利用群体相对策略优化（Group Relative Policy Optimization）进一步优化，Doc-$V^*$ 在回答准确性与证据寻找效率之间取得了平衡。在五个基准测试中，Doc-$V^*$ 的表现优于开源基准模型，并接近私有模型，在域外性能上比 RAG 基准提升了高达 47.9%。其他结果表明，有效的证据聚合源于选择性注意力，而非增加输入页数。

## Abstract
Multi-page Document Visual Question Answering requires reasoning over semantics, layouts, and visual elements in long, visually dense documents. Existing OCR-free methods face a trade-off between capacity and precision: end-to-end models scale poorly with document length, while visual retrieval-based pipelines are brittle and passive. We propose Doc-$V^*$, an \textbf{OCR-free agentic} framework that casts multi-page DocVQA as sequential evidence aggregation. Doc-$V^*$ begins with a thumbnail overview, then actively navigates via semantic retrieval and targeted page fetching, and aggregates evidence in a structured working memory for grounded reasoning. Trained by imitation learning from expert trajectories and further optimized with Group Relative Policy Optimization, Doc-$V^*$ balances answer accuracy with evidence-seeking efficiency. Across five benchmarks, Doc-$V^*$ outperforms open-source baselines and approaches proprietary models, improving out-of-domain performance by up to \textbf{47.9\%} over RAG baseline. Other results reveal effective evidence aggregation with selective attention, not increased input pages.

#### 1. 核心问题与整体含义（研究动机和背景）
多页文档视觉问答（DocVQA）要求模型能够同时处理长文档中的语义、布局和视觉元素（如表格、图表）。现有的无 OCR（OCR-free）方法面临**容量与精度**的权衡：
*   **端到端模型（E2E）：** 难以扩展到超长文档，受限于计算成本、上下文长度限制以及“信息迷失在中间”的效应。
*   **检索增强生成（RAG）：** 属于被动处理，容易受到检索错误的影响，且难以处理需要多跳推理的复杂问题。
*   **研究动机：** 模拟人类阅读行为——先通过缩略图获取全局结构，再主动寻找、验证并整合证据。论文旨在提出一个主动的、智能体化的框架，将 DocVQA 转化为一个**序列化的证据聚合过程**。

#### 2. 方法论：核心思想与技术细节
Doc-V* 是一个基于多模态大语言模型（MLLM）的智能体框架，其核心流程如下：
*   **全局缩略图概览（Global Thumbnail Overview）：** 将文档页面调整为低分辨率缩略图并排列成网格，作为初始观察。这提供了文档类型、布局和图表分布的全局先验，且显著降低了视觉 Token 的消耗（压缩比约 10-12 倍）。
*   **动作空间（Action Space）：**
    1.  `retrieval_page`: 类似于“Ctrl+F”，通过语义查询检索最相关的页面。
    2.  `fetch_page`: 根据缩略图观察或上下文，直接获取特定页码的高分辨率图像。
    3.  `answer`: 当证据充足时终止交互并输出答案。
*   **结构化视觉推理：** 采用 ReAct 风格，输出包含 `<think>`（分析、计划、总结）和 `<action>`。引入**结构化工作记忆（Working Memory）**，将每一步的总结串联，防止长序列交互中的遗忘。
*   **两阶段训练策略：**
    1.  **监督微调（SFT）：** 使用 GPT-4o 蒸馏出的高质量交互轨迹进行冷启动。
    2.  **强化学习（GRPO）：** 采用群体相对策略优化（Group Relative Policy Optimization），通过奖励函数平衡回答准确性、证据召回率和格式规范性，优化智能体的探索效率。

#### 6. 主要结论与发现
*   **性能领先：** Doc-V* 在开源模型中表现最优，甚至在某些指标上超越了 GPT-4o 和 Claude-3.7。

#### 7. 优点
*   **主动感知：** 改变了传统 RAG 的被动模式，赋予模型主动探索和验证的能力。
*   **效率平衡：** 通过缩略图和按需获取高分辨率页面，在计算成本和感知精度之间找到了平衡点。
*   **强化学习优化：** 成功将 GRPO 应用于多模态智能体，显著提升了决策的逻辑性和效率。

*   **推理成本：** 虽然比全量输入节省资源，但多轮交互带来的推理延迟仍高于单次 RAG 检索。
