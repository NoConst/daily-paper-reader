## 摘要
深度研究智能体 (DRA) 将大语言模型 (LLM) 的推理能力与外部工具相结合。记忆系统使 DRA 能够利用历史经验，这对于高效推理和自主演化至关重要。现有方法依赖于从记忆中检索相似轨迹来辅助推理，但面临记忆演化效率低下以及存储和检索成本不断增加的关键局限。为解决这些问题，我们提出了一种新型的记忆智能体 (MIA) 框架，该框架由“管理者-规划者-执行者” (Manager-Planner-Executor) 架构组成。记忆管理者是一个非参数化记忆系统，可以存储压缩后的历史搜索轨迹。规划者是一个参数化记忆智能体，能够为问题生成搜索计划。执行者是另一个在搜索计划指导下进行信息搜索和分析的智能体。为了构建 MIA 框架，我们首先采用交替强化学习范式来增强规划者与执行者之间的协作。此外，我们使规划者能够在测试时学习过程中持续演化，在不中断推理过程的情况下，随推理同步进行实时更新。此外，我们在参数化记忆和非参数化记忆之间建立了双向转换循环，以实现高效的记忆演化。最后，我们引入了反思和无监督判断机制，以提升在开放世界中的推理和自我演化能力。在 11 个基准测试上的广泛实验证明了 MIA 的优越性。

## Abstract
Deep research agents (DRAs) integrate LLM reasoning with external tools. Memory systems enable DRAs to leverage historical experiences, which are essential for efficient reasoning and autonomous evolution. Existing methods rely on retrieving similar trajectories from memory to aid reasoning, while suffering from key limitations of ineffective memory evolution and increasing storage and retrieval costs. To address these problems, we propose a novel Memory Intelligence Agent (MIA) framework, consisting of a Manager-Planner-Executor architecture. Memory Manager is a non-parametric memory system that can store compressed historical search trajectories. Planner is a parametric memory agent that can produce search plans for questions. Executor is another agent that can search and analyze information guided by the search plan. To build the MIA framework, we first adopt an alternating reinforcement learning paradigm to enhance cooperation between the Planner and the Executor. Furthermore, we enable the Planner to continuously evolve during test-time learning, with updates performed on-the-fly alongside inference without interrupting the reasoning process. Additionally, we establish a bidirectional conversion loop between parametric and non-parametric memories to achieve efficient memory evolution. Finally, we incorporate a reflection and an unsupervised judgment mechanisms to boost reasoning and self-evolution in the open world. Extensive experiments across eleven benchmarks demonstrate the superiority of MIA.

    1.  **注意力稀释**：过长的上下文干扰模型对当前问题的理解。
    2.  **噪声干扰**：无关或弱相关的历史信息降低推理质量。
    3.  **存储与检索成本高**：随着任务增加，存储压力和计算开销呈指数级增长。
    4.  **缺乏策略复用**：现有记忆多关注“事实是什么”，而非“如何搜索和推理”（过程导向记忆）。
*   **核心目标**：通过解耦记忆管理、计划制定与任务执行，实现记忆的高效压缩、参数化内化以及在无监督环境下的持续进化。

### 2. 方法论
MIA 采用了 **Manager-Planner-Executor（管理者-规划者-执行者）** 架构：
*   **核心思想**：将记忆分为**非参数化记忆**（存储压缩后的工作流轨迹）和**参数化记忆**（通过训练将经验内化到规划者的模型参数中）。
*   **关键技术细节**：
    *   **记忆管理者 (Manager)**：负责将冗长的搜索轨迹压缩为结构化的“工作流摘要”，并根据语义相似度、价值奖励和频率奖励进行混合检索。
    *   **规划者 (Planner)**：作为认知核心，利用检索到的记忆生成分步搜索计划，并具备“反思-重规划”能力。
    *   **执行者 (Executor)**：负责具体的工具调用（如网页搜索、图像搜索）并反馈结果。
*   **算法流程**：
    1.  **两阶段交替强化学习 (Alternating RL)**：基于 GRPO 算法。第一阶段训练执行者理解并遵循计划；第二阶段训练规划者吸收记忆并优化计划生成。
    2.  **测试时学习 (Test-Time Learning, TTL)**：在推理阶段，智能体同步进行探索、存储和参数更新，实现“边做边学”。
    3.  **无监督自演化**：引入模拟学术评审的 **Reviewer-Area Chair 机制**，通过逻辑、事实、格式三个维度的交叉评审，在没有标准答案的情况下为强化学习提供奖励信号。

### 6. 主要结论与发现
*   **性能飞跃**：MIA 显著提升了现有最强模型的表现。例如，使 GPT-5.4 在 LiveVQA 上提升了 9%，在 HotpotQA 上提升了 6%。

*   **记忆效率**：相比长上下文记忆，MIA 的压缩工作流和参数化内化机制有效减少了噪声，提高了工具调用的成功率。

### 7. 优点
*   **架构创新**：Manager-Planner-Executor 的解耦设计非常符合人类“经验总结-制定计划-动手执行”的认知逻辑。
*   **双向记忆转换**：实现了非参数化（显式）与参数化（隐式）记忆的互补，解决了存储瓶颈。
*   **无监督机制**：提出的 Reviewer-AC 评审框架为解决 LLM 训练中的“标签稀缺”问题提供了新思路。
*   **实时性**：TTL 机制允许模型在推理过程中进化，无需离线重新训练。

### 8. 不足与局限
*   **任务局限性**：目前主要集中在“深度研究”（信息检索与推理）任务，对于需要实时物理交互或极高创造性的任务（如艺术创作）覆盖不足。
*   **计算开销**：虽然减少了存储成本，但 TTL 过程中的多路采样（Rollout）和在线训练对推理时的算力有一定额外要求。
*   **偏差风险**：无监督演化依赖于上位模型（如 Qwen3-32B）的判断力，若上位模型存在系统性偏见，可能会引导智能体进入错误的进化方向。
