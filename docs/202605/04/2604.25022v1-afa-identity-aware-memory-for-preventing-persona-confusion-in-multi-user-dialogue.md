## 摘要
当多人共用一个语音助手时，系统会混淆他们的历史记录：一位居住者的偏好可能会泄露到另一位居住者的回复中，从而削弱实用性和信任。我们将这种故障模式称为“人格混淆”（persona confusion），并证明了在共享环境中部署当今的单用户对话系统时，这是一个可衡量的问题。我们提出了自适应朋友智能体（Adaptive Friend Agent, AFA），这是一个模块化框架，它将基于语音的说话人识别与每个用户的存储记忆相结合，以实现跨多用户的身份感知个性化对话。为了支持训练和评估，我们构建了 PAT（Personalized Agent chaT），这是一个包含 58,289 个基于人格的对话轮次的合成数据集，涵盖了 133 个用户档案和 12 个真实场景。我们在标准回复质量基准测试中评估了 AFA 在五个大语言模型（LLM）后端上的表现，其中在 PAT 上微调的 LLaMA-2-70B 模型取得了最高的整体性能。为了直接衡量人格混淆的预防效果，我们引入了一种交错多用户评估协议，并提出了一种新指标——人格归属准确率（Persona Attribution Accuracy, PAA），证明了身份感知路由将 PAA 从 35.7% 提高到了 61.3%。人工评估证实，标注者认为启用路由的回复具有显著更高的个性化程度。我们的结果表明，身份感知用户路由是防止多用户对话系统中人格混淆的关键组件。

## Abstract
When multiple people share a single voice assistant, the system conflates their histories: one resident's preferences can leak into another's responses, eroding utility and trust. We call this failure mode persona confusion, and we show it is a measurable problem in today's single-user dialogue systems when deployed in shared environments. We present the Adaptive Friend Agent (AFA), a modular framework that combines voice-based speaker identification with per-user memory stores to enable identity-aware, personalized dialogue across multiple users. To support training and evaluation, we construct PAT (Personalized Agent chaT), a synthetic dataset of 58,289 persona-grounded dialogue turns spanning 133 user profiles and 12 real-world scenarios. We evaluate AFA across five LLM back-ends in a standard response-quality benchmark, with a LLaMA-2-70B model fine-tuned on PAT achieving the highest overall performance. To directly measure persona confusion prevention, we introduce an interleaved multi-user evaluation protocol with a novel metric, Persona Attribution Accuracy (PAA), demonstrating that identity-aware routing improves PAA from 35.7% to 61.3%. Human evaluation confirms annotators perceive significantly higher personalization in routing-enabled responses. Our results establish that identity-aware user routing is the critical component for preventing persona confusion in multi-user conversational systems.

*   **研究动机**：这种“人格混淆”（Persona Confusion）不仅是技术错误，更会损害系统的实用性和用户信任。研究旨在通过身份感知的记忆路由机制，确保系统能为每个用户提供独立且一致的个性化体验。

### 2. 核心方法论：AFA 框架
AFA（Adaptive Friend Agent）是一个模块化框架，其核心思想是**“身份识别 + 记忆隔离 + 动态同步”**。
*   **说话人识别（Speaker ID）**：利用 SpeechBrain 的 ECAPA-TDNN 模型提取语音嵌入（Voice Embeddings），通过余弦相似度匹配用户 ID。
*   **身份感知路由与隔离存储**：系统为每个用户维护独立的向量数据库。根据识别出的用户 ID，将查询路由至该用户的专属存储区，检索其人格属性和历史对话。
*   **动态人格同步器（Persona Synchronizer）**：利用 GPT-4o 从每轮对话中提取六类属性（人口统计、职业、动机价值观、决策风格、偏好、情感触发点），实时更新用户画像。
*   **自适应回复生成**：将检索到的特定用户人格、历史背景与当前查询拼接成 Prompt，输入 LLM 生成回复。

### 3. 实验设计
*   **数据集 (PAT)**：研究者构建了 **Personalized Agent chaT (PAT)** 数据集，包含 58,289 轮对话，涵盖 133 个独特人格和 12 个真实场景（如项目规划、情感支持、旅游规划等）。

    *   **评估指标**：除了 BLEU/ROUGE 等文本质量指标，引入了 **PAA（人格归属准确率）** 和 **Persona Margin**，用于衡量回复是否准确偏向当前用户而非混淆对象。

*   **充分性评价**：实验设计较为全面，不仅涵盖了生成质量，还专门针对“多用户干扰”这一核心痛点设计了消融实验和新指标。通过对比“无路由”和“有路由”的表现，客观证明了身份识别在个性化系统中的决定性作用。

### 6. 主要结论与发现
*   **路由是关键**：身份感知路由将人格归属准确率（PAA）从 **35.7% 显著提升至 61.3%**。
*   **微调效果显著**：在 PAT 数据集上微调的 LLaMA-2-70B 在多项指标上接近甚至超过了更大的闭源模型（如 GPT-4o），证明了高质量领域数据的价值。
*   **人格混淆是真实存在的**：在无路由的基准测试中，PAA 低于 50%（随机水平），说明系统会主动向错误的用户人格漂移。
*   **自适应性**：虽然短期内固定人格已能提供较好效果，但自适应同步机制在长期部署中对捕捉用户偏好演变至关重要。

### 7. 优点与亮点
*   **新指标 PAA**：提出了一种控制话题偏差的人格归属度量方法，解决了传统指标无法区分“话题相关性”与“人格一致性”的问题。
*   **数据集贡献**：PAT 数据集填补了多用户、多场景、人机交互个性化对话数据的空白。
*   **模块化设计**：AFA 框架不绑定特定 LLM 或声纹模型，具有很强的可扩展性和工业应用潜力。

### 8. 不足与局限
*   **数据合成性**：PAT 数据集主要由 LLM 生成，可能与真实人类对话的分布存在差异。
*   **声纹环境模拟**：实验使用了合成语音，未充分测试真实世界中的环境噪音、多人同时说话（重叠声）或远场拾音对识别准确率的影响。
*   **冷启动挑战**：在完全没有预设画像的情况下，系统需要约 20 轮对话才能达到较低的识别精度，对于即插即用的体验仍有提升空间。
*   **隐私风险**：虽然采用了数据隔离，但声纹作为生物识别信息，在共享环境中的存储和合规性仍需进一步探讨。
