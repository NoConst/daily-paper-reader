## 摘要
我们提出了 Action Agent，这是一个两阶段框架，将智能体导航视频生成与流约束扩散控制相结合，用于多具身机器人导航。在第一阶段，大语言模型（LLM）作为编排模块，负责选择视频扩散模型、通过迭代验证优化提示词，并积累跨任务记忆，从而根据语言和图像输入合成物理上合理的视角导航视频。在 50 个导航任务中，这使视频生成的成功率从 35%（单次尝试）提升至 86%。在第二阶段，我们引入了 FlowDiT，这是一种流约束扩散 Transformer（Flow-Constrained Diffusion Transformer），它利用动作空间去噪扩散将优化的目标视频和语言指令转换为连续的速度指令。FlowDiT 集成了 DINOv2 视觉特征、用于自我运动表示的学习光流，以及用于语义停止的 CLIP 语言嵌入。我们在 RECON 户外导航数据集上进行预训练，并在 Isaac Sim 中采集的 203 个 Unitree G1 人形机器人片段上进行微调，以校准速度动力学。一个仅包含 43M 参数的检查点在仿真中实现了 73.2% 的导航成功率，在开环执行下，在未见过的室内环境中实现了真实 Unitree G1 机器人 64.7% 的任务完成率，同时运行频率达到 40-47 Hz。我们在三种具身形态上评估了 Action Agent：Unitree G1 人形机器人（真实硬件）、无人机和轮式移动机器人（Isaac Sim），证明了将轨迹想象与执行解耦为语言引导的导航提供了一种可扩展且具身感知的范式。

## Abstract
We present Action Agent, a two-stage framework that unifies agentic navigation video generation with flow-constrained diffusion control for multi-embodiment robot navigation. In Stage I, a large language model (LLM) acts as an orchestration module that selects video diffusion models, refines prompts through iterative validation, and accumulates cross-task memory to synthesize physically plausible first-person navigation videos from language and image inputs. This increases video generation success from 35% (single-shot) to 86% across 50 navigation tasks. In Stage II, we introduce FlowDiT, a Flow-Constrained Diffusion Transformer that converts optimized goal videos and language instructions into continuous velocity commands using action-space denoising diffusion. FlowDiT integrates DINOv2 visual features, learned optical flow for ego-motion representation, and CLIP language embeddings for semantic stopping. We pretrain on the RECON outdoor navigation dataset and fine-tune on 203 Unitree G1 humanoid episodes collected in Isaac Sim to calibrate velocity dynamics. A single 43M-parameter checkpoint achieves 73.2% navigation success in simulation and 64.7% task completion on a real Unitree G1 in unseen indoor environments under open-loop execution, while operating at 40--47 Hz. We evaluate Action Agent across three embodiments: a Unitree G1 humanoid (real hardware), a drone, and a wheeled mobile robot (Isaac Sim), demonstrating that decoupling trajectory imagination from execution yields a scalable and embodiment-aware paradigm for language-guided navigation.

*   **研究动机**：传统的机器人导航依赖复杂的 SLAM 和路径规划，难以适应环境变化；而端到端的视觉-语言-动作（VLA）模型虽然强大，但往往是“黑盒”，计算开销巨大，且容易产生空间推理幻觉。
*   **核心思想**：将导航任务解耦为“**轨迹想象**”和“**动作执行**”两个阶段。首先通过视频生成模型“排练”出一段视觉上的导航参考视频（视觉中间表示，VIR），然后再由轻量级的控制模型将视频转化为具体的物理动作。

### 2. 方法论
该框架分为两个核心阶段：
*   **第一阶段：智能体轨迹想象（Agentic Trajectory Imagination）**
    *   **核心流程**：LLM 作为编排器，协调视觉语言模型（Qwen3-VL）构建提示词，调用视频生成模型（WAN 2.2 或 LTX-Video）合成第一视角视频，并由推理验证器（Cosmos-Reason1）根据指令遵循度（PA）、物理合理性（PP）和视觉质量（VQ）进行评分。
    *   **迭代优化**：如果评分未达标，LLM 会根据反馈和“跨任务记忆”（长期、短期和静态规则记忆）迭代修改提示词，直到生成物理上合理的视频。
*   **第二阶段：流约束扩散 Transformer（FlowDiT）**
    *   **核心技术**：使用扩散 Transformer（DiT）架构，将参考视频转化为连续的速度指令 $(v_x, v_y, \omega)$。
    *   **多模态调节**：模型输入集成了 DINOv2 视觉特征、专门学习的**光流（Optical Flow）**特征（用于捕捉自我运动和深度信息）以及 CLIP 语言嵌入（用于识别语义停止点）。
    *   **执行模式**：支持开环（仅靠参考视频）和闭环（结合实时相机输入）控制，运行频率高达 40–47 Hz。

### 6. 主要结论与发现
*   **智能体迭代的价值**：通过 LLM 迭代优化，视频生成的成功率从单次尝试的 35% 提升至 **86%**。
*   **性能表现**：FlowDiT 在仿真中达到 73.2% 的成功率；在真实人形机器人开环测试中，任务完成率为 **64.7%**。

### 7. 优点
*   **解耦架构**：将高层语义推理（视频生成）与底层物理控制（扩散策略）分开，既保留了生成式模型的想象力，又保证了控制的实时性。
*   **极高的效率**：43M 的参数量使其非常适合部署在计算资源有限的机器人边缘端。
*   **具身感知**：通过第一阶段的验证循环，生成的视频能考虑到机器人的物理约束。

### 8. 不足与局限
*   **尺度歧义（Scale Ambiguity）**：在真实机器人开环执行时，生成的视频像素运动与实际物理位移之间存在尺度误差，导致轨迹漂移。
*   **精度限制**：在极度狭窄的空间（如窄门框）中，厘米级的避障精度仍有待提高。
*   **规划时效**：受限于当前视频生成模型，单次规划的视频长度仅为 5–15 秒，限制了长距离导航能力。
