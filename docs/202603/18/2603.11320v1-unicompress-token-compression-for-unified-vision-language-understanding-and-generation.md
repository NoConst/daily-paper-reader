## 摘要
统一模型旨在通过将图像编码为离散 token，并在单个自回归框架内将其与文本共同处理，从而同时支持理解与生成任务。这种统一设计提供了架构上的简洁性和跨模态协同效应，有助于实现共享参数化、一致的训练目标以及模态间的无缝迁移。然而，此类模型所需的大量视觉 token 带来了巨大的计算和内存开销，这种低效性直接阻碍了其在具身智能系统等资源受限场景中的部署。在这项工作中，我们提出了一种统一的 token 压缩算法 UniCompress，该算法在显著减少视觉 token 数量的同时，保持了在图像理解和生成任务上的性能。我们的方法引入了一种由可学习的全局元 token 引导的插件式压缩与解压缩机制。该框架轻量且模块化，能够高效集成到现有模型中而无需进行完整的重新训练。实验结果表明，我们的方法将图像 token 减少了多达 4 倍，在推理延迟和训练成本方面取得了显著收益，且仅造成极小的性能下降，这证明了高效 token 统一建模在现实世界多模态应用中的前景。

## Abstract
Unified models aim to support both understanding and generation by encoding images into discrete tokens and processing them alongside text within a single autoregressive framework. This unified design offers architectural simplicity and cross-modal synergy, which facilitates shared parameterization, consistent training objectives, and seamless transfer between modalities. However, the large number of visual tokens required by such models introduces substantial computation and memory overhead, and this inefficiency directly hinders deployment in resource constrained scenarios such as embodied AI systems. In this work, we propose a unified token compression algorithm UniCompress that significantly reduces visual token count while preserving performance on both image understanding and generation tasks. Our method introduces a plug-in compression and decompression mechanism guided with learnable global meta tokens. The framework is lightweight and modular, enabling efficient integration into existing models without full retraining. Experimental results show that our approach reduces image tokens by up to 4 times, achieves substantial gains in inference latency and training cost, and incurs only minimal performance degradation, which demonstrates the promise of token-efficient unified modeling for real world multimodal applications.

#### 1. 核心问题与整体含义（研究动机和背景）
当前的统一视觉语言模型（Unified VLMs）通过将图像编码为离散 Token，在单一自回归框架内实现图像理解与生成。然而，这类模型面临严重的**Token 效率瓶颈**：
*   **计算开销大**：为了保证生成质量，通常需要大量的视觉 Token（如 1024 个），导致内存占用高、推理延迟大，难以在具身智能等资源受限场景部署。
*   **压缩矛盾**：简单的下采样或剪枝虽能维持理解性能，但会严重破坏图像生成的空间一致性和细节，导致生成质量大幅下降（实验显示下降超过 15%）。
*   **迁移成本高**：重新训练一个高效的 Tokenizer 往往需要从头训练底层的语言模型（LLM），成本极高。

**UniCompress** 的目标是提出一种插件式、轻量化的压缩方案，在不显著损失性能的前提下，将视觉 Token 数量减少 4 倍，并兼容现有模型。

#### 2. 方法论：核心思想与关键技术
UniCompress 引入了一个由**全局元标记（Global Meta Tokens）**引导的插件式压缩与解压缩机制，核心组件包括：
*   **全局标记提取（Global Token Extraction）**：利用 Cross-Attention 机制，通过一小组可学习的元标记（Meta Queries）从原始图像特征中提取全局上下文信息。这些全局 Token 充当语义锚点，捕捉宏观布局。
*   **池化压缩器（Pooling-based Compressor）**：对原始 $H \times W$ 的 Token 网格进行非重叠平均池化（如 $2 \times 2$），将序列长度缩减为原来的 $1/s^2$。
*   **自回归解压缩器（Autoregressive Decompressor）**：这是保证生成质量的关键。在生成阶段，LLM 输出压缩后的 Token，解压缩器在全局 Token 的引导下，通过自回归方式将紧凑表示恢复为高保真的密集 Token 网格，从而修复局部纹理和边界。
*   **两阶段训练流程**：
    1.  **Tokenizer 侧训练**：冻结 LLM，仅训练压缩/解压缩模块和全局提取器，优化重构损失。
    2.  **LLM 侧训练**：冻结 Tokenizer 侧，在压缩后的 Token 空间上对 LLM 进行轻量级微调。

*   **端到端加速**：不仅减少了存储，还通过缩短序列长度直接提升了自回归生成的推理速度。
*   **设计精巧**：引入全局元标记解决了传统压缩方法在生成任务上的“短板”。

*   **模型规模覆盖**：实验主要集中在 1B 规模的轻量级 LLM 上，尚未在大规模参数（如 7B, 70B）模型上验证压缩带来的增益是否会随规模效应改变。
