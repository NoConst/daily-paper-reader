## 摘要
为了维持连贯的长期交互，大语言模型（LLM）智能体必须处理获取新信息与保留先验知识之间的矛盾。当前统一的流式记忆系统虽然便于上下文更新，但仍易受瞬态噪声的干扰。相反，离散的结构化记忆架构提供了稳健的知识保留能力，但往往难以适应不断演变的叙述。为解决这一问题，我们提出了 GAM，这是一种分层图结构代理记忆框架，它通过显式地将记忆编码与巩固解耦，从而有效解决快速上下文感知与稳定知识保留之间的冲突。通过将正在进行的对话隔离在事件演进图中，并仅在发生语义偏移时将其整合到主题关联网络中，我们的方法在保持长期一致性的同时最大限度地减少了干扰。此外，我们引入了一种图引导的多因素检索策略，以提高上下文的精准度。在 LoCoMo 和 LongDialQA 上的实验表明，我们的方法在推理准确性和效率方面均持续优于现有的基准模型。

## Abstract
To sustain coherent long-term interactions, Large Language Model (LLM) agents must navigate the tension between acquiring new information and retaining prior knowledge. Current unified stream-based memory systems facilitate context updates but remain vulnerable to interference from transient noise. Conversely, discrete structured memory architectures provide robust knowledge retention but often struggle to adapt to evolving narratives. To address this, we propose GAM, a hierarchical Graph-based Agentic Memory framework that explicitly decouples memory encoding from consolidation to effectively resolve the conflict between rapid context perception and stable knowledge retention. By isolating ongoing dialogue in an event progression graph and integrating it into a topic associative network only upon semantic shifts, our approach minimizes interference while preserving long-term consistency. Additionally, we introduce a graph-guided, multi-factor retrieval strategy to enhance context precision. Experiments on LoCoMo and LongDialQA indicate that our method consistently outperforms state-of-the-art baselines in both reasoning accuracy and efficiency.

### 2. 方法论：核心思想与技术细节
GAM 借鉴了人类睡眠驱动的记忆巩固机制，将记忆生命周期分为两个阶段：

*   **分层图记忆架构：**
    *   **事件推进图 ($G_{event}$):** 局部层，用于捕获实时对话的原子单元和时间/因果依赖，起到“写隔离”缓冲作用。
    *   **主题关联网络 ($G_{topic}$):** 全局层，存储高层语义集群和抽象主题，通过语义边连接。
    *   **跨层关联 ($E_{cross}$):** 将全局主题节点链接到具体的历史事件快照。
*   **基于状态的记忆巩固机制：**
    *   **情节缓冲状态：** 智能体在局部图中快速记录对话细节。
    *   **语义巩固状态：** 当检测到“语义边界”（通过 LLM 判定话题切换）时，触发巩固。将局部图总结为语义节点，并整合进全局网络。
*   **图引导的多因素检索策略：**
    *   采用“自顶向下”的遍历：先定位核心主题，再钻取到具体的事件细节。
    *   **多因素重排序：** 结合语义相关性、**时间信号**、**置信度**和**角色信息**（针对多用户对话）进行综合评分。

### 6. 主要结论与发现
*   **性能领先：** GAM 在推理准确性（F1 和 BLEU-1 分数）上显著优于 Mem0 和 MemoryOS。在 Qwen2.5-7B 上，其时间推理 F1 分数比 Mem0 高出 18%。
*   **解耦的有效性：** 实验证明，通过物理隔离局部缓冲和全局存储，可以有效防止“记忆污染”。
*   **效率优势：** 尽管使用了图结构，但通过“粗到精”的候选选择策略，GAM 在保持高精度的同时，降低了 Token 成本和延迟。

### 7. 优点与亮点
*   **架构创新：** 显式解耦编码与巩固过程，解决了流式记忆的污染问题。
*   **检索精准：** 引入角色和时间等多维度因子，特别适合处理多人、长跨度的复杂对话。
*   **即插即用：** 无需微调模型，对不同规模的开源和闭源模型均有显著提升。
*   **动态适应：** 相比静态知识图谱，GAM 能随对话进程动态调整结构。

### 8. 不足与局限
*   **模态限制：** 目前仅支持文本模态，无法处理视觉或音频等非语言信号（尽管作者在附录中做了多模态代理实验，但原生支持尚缺）。

*   **复杂性：** 相比简单的向量检索，图结构的维护和多层遍历增加了系统的实现复杂度。
