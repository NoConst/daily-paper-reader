## 摘要
当多人共用一个语音助手时，系统会混淆他们的历史记录：一个居住者的偏好可能会泄露到另一个人的回复中，从而损害实用性和信任。我们将这种故障模式称为“人格混淆”（persona confusion），并证明在共享环境中部署当今的单用户对话系统时，这是一个可衡量的问题。我们提出了自适应朋友代理（Adaptive Friend Agent, AFA），这是一个模块化框架，它将基于语音的说话人识别与每个用户的存储记忆相结合，以实现跨多用户的身份感知个性化对话。为了支持训练和评估，我们构建了 PAT（Personalized Agent chaT），这是一个包含 58,289 个基于人格的对话轮次的合成数据集，涵盖了 133 个用户配置文件和 12 个真实场景。我们在标准回复质量基准测试中评估了 AFA 在五个大语言模型（LLM）后端上的表现，其中在 PAT 上微调的 LLaMA-2-70B 模型获得了最高的整体性能。为了直接衡量人格混淆的预防效果，我们引入了一种交错多用户评估协议和一种新的指标——人格归属准确率（Persona Attribution Accuracy, PAA），证明了身份感知路由将 PAA 从 35.7% 提高到了 61.3%。人工评估证实，标注者认为启用路由的回复具有显著更高的个性化程度。我们的结果表明，身份感知用户路由是防止多用户对话系统中人格混淆的关键组件。

## Abstract
When multiple people share a single voice assistant, the system conflates their histories: one resident's preferences can leak into another's responses, eroding utility and trust. We call this failure mode persona confusion, and we show it is a measurable problem in today's single-user dialogue systems when deployed in shared environments. We present the Adaptive Friend Agent (AFA), a modular framework that combines voice-based speaker identification with per-user memory stores to enable identity-aware, personalized dialogue across multiple users. To support training and evaluation, we construct PAT (Personalized Agent chaT), a synthetic dataset of 58,289 persona-grounded dialogue turns spanning 133 user profiles and 12 real-world scenarios. We evaluate AFA across five LLM back-ends in a standard response-quality benchmark, with a LLaMA-2-70B model fine-tuned on PAT achieving the highest overall performance. To directly measure persona confusion prevention, we introduce an interleaved multi-user evaluation protocol with a novel metric, Persona Attribution Accuracy (PAA), demonstrating that identity-aware routing improves PAA from 35.7% to 61.3%. Human evaluation confirms annotators perceive significantly higher personalization in routing-enabled responses. Our results establish that identity-aware user routing is the critical component for preventing persona confusion in multi-user conversational systems.

*   **研究动机**：这种人格混淆不仅降低了系统的实用性，还会损害用户信任。研究旨在通过引入身份识别和隔离的记忆机制，确保系统能为每个用户提供精准、隔离且持续进化的个性化服务。

### 2. 方法论：AFA 框架
AFA（Adaptive Friend Agent）是一个模块化框架，其核心思想是“**身份感知路由**”。
*   **说话人识别（Speaker ID）**：利用 SpeechBrain 的 ECAPA-TDNN 模型提取语音嵌入（Embeddings），通过余弦相似度匹配用户 ID。
*   **动态用户配置文件存储**：为每个用户建立独立的向量数据库。
    *   **临时表**：存储最近 10 次对话的原始上下文。
    *   **永久表**：存储由 GPT-4o 生成的会话摘要，平衡存储效率与长短期记忆。
*   **人格同步器（Persona Synchronizer）**：从对话中提取 6 类属性（人口统计、职业、价值观、决策风格、偏好、情感触发点），动态更新用户画像。
*   **自适应回复生成器**：将识别出的用户 ID 作为路由，从该用户的专属记忆库中检索 Top-3 相关历史，连同当前画像注入 LLM 提示词中生成回复。

*   **数据集 (PAT)**：研究者构建了 **Personalized Agent chaT (PAT)** 数据集，包含 58,289 轮对话，涵盖 133 个独特人格和 12 个真实场景（如项目规划、情感支持、旅游规划等）。

### 6. 主要结论与发现
*   **路由是关键**：身份感知路由将人格归属准确率（PAA）从 **35.7% 提升至 61.3%**。
*   **微调效果显著**：在 PAT 数据集上微调的 LLaMA-2-70B 在多项指标上接近甚至超过了更大的闭源模型（如 GPT-4o）。
*   **自适应性**：虽然自适应画像在短期对话中提升有限（相比恒定画像），但在长期部署中对于捕捉用户偏好演变至关重要。
*   **人格混淆真实存在**：在无路由基准下，PAA 低于随机水平（50%），说明系统会主动向错误的用户偏好漂移。

### 7. 优点与亮点
*   **新指标 PAA**：解决了传统 NLP 指标无法衡量“人格归属正确性”的问题，通过减去基准相似度控制了话题偏见。
*   **模块化设计**：声纹识别、记忆存储和生成模块解耦，易于升级（如更换更强的声纹模型或 LLM）。
*   **高质量数据集**：PAT 数据集填补了多用户、多会话、人机交互个性化数据集的空白。

### 8. 不足与局限
*   **合成数据偏差**：PAT 数据集完全由 LLM 生成，可能与真实人类对话的分布存在差异。
*   **声纹识别环境**：实验使用了合成语音，未充分考虑真实世界中的背景噪音、多人同时说话（重叠声）或远场拾音的挑战。
*   **冷启动问题**：实验发现，如果没有初始画像（Onboarding），仅靠对话积累画像的 PAA 仅为 41%，说明系统对初始信息的依赖较强。
*   **隐私风险**：虽然实现了数据隔离，但声纹作为生物识别信息的存储和处理在实际应用中面临严格的隐私监管。
